# 回测验收和防过拟合手册

## 先看四件事

| 指标 | 为什么重要 |
|---|---|
| 最大回撤 | 决定老大能不能真实拿住 |
| 样本外表现 | 决定是不是只拟合了过去 |
| 换手和成本 | 决定纸面收益能不能留下 |
| 参数稳定性 | 决定策略是不是只有一个点漂亮 |

## 最低验收线

| 项目 | 最低要求 |
|---|---|
| 训练/验证/测试切分 | 必须有，测试期至少 2 年更稳 |
| 交易成本 | 佣金、印花税、滑点都计入 |
| 最大回撤 | 必须在账户能承受范围内 |
| 年度分布 | 不只靠某一年贡献收益 |
| 换手率 | 小票高换手要做 2x/3x 成本敏感性 |
| 样本外衰减 | 测试期 Sharpe 不应严重塌陷 |
| 对照组 | 至少对比等权、指数、简单动量或红利基准 |

## 防过拟合 Checklist

| 检查 | PASS 标准 |
|---|---|
| 规则先冻结 | 回测前已有 strategy spec |
| 参数数量 | 参数少且有经济含义 |
| 训练/验证/测试 | 日期清楚，测试期未参与调参 |
| 分年表现 | 牛、熊、震荡年份都看过 |
| 成本敏感性 | 2x/3x 成本下不直接失效 |
| 参数扰动 | 相邻参数仍有效 |
| 股票池扰动 | 换成中证500/1000或全A后不完全消失 |
| 因子方向 | 果仁/分组回测有单调性 |
| 流动性 | 成交额、停牌、涨跌停过滤 |
| 日志 | 每次改动说明“为什么改” |

## 结果分级

| 等级 | 含义 | 下一步 |
|---|---|---|
| A | 样本外稳定、回撤可控、成本敏感性通过 | 模拟盘或小仓 |
| B | 有方向，但回撤/换手/样本外有短板 | 继续降频、降换手、加风控 |
| C | 样本内漂亮，样本外明显衰减 | 暂停实盘，只保留研究 |
| D | 依赖未来函数/高过拟合/不可交易 | 废弃或重写 |

## 老大式实盘前问题

实盘前必须能回答：

1. 这策略最怕什么市场？
2. 连亏多久算失效？
3. 最大回撤到多少我会停？
4. 我有没有能力按规则执行？
5. 如果小仓亏了，是策略错还是执行错？

回答不了，就先别上实盘。
