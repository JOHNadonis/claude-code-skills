# A 股小盘股提示词模板

## 模板一：高质量小盘公司筛选

```text
请扮演资深 A 股量化分析师，基于最新交易日行情、最近三年年报、最近四个季度财报、公司公告和交易所监管信息，筛选 A 股小盘公司。

市场范围：
沪深主板、创业板、科创板；北交所仅在流动性满足条件时纳入。

硬性排除：
ST、*ST、退市整理、暂停上市、近一年被证监会立案调查、审计意见非标、重大资金占用、重大违规担保、重大未决诉讼、控股股东高比例质押且无缓释方案。

筛选条件：
1. 总市值介于人民币 [下限] 亿至 [上限] 亿元。
2. 资产负债率低于 30%，流动比率大于 2。
3. 过去三年加权 ROE 均高于 15%，扣非净利润为正。
4. 经营现金流净额连续三年为正，且经营现金流净额 / 净利润大于 0.8。
5. 连续三年现金分红，优先选择每股分红或分红率稳定上升的公司。
6. 近 20 个交易日日均成交额不低于 [金额] 万元。

输出前 10 家公司：
- 股票代码和名称
- 所属申万行业或主业标签
- 市值
- 核心财务指标
- 分红情况
- 细分行业护城河
- 主要风险
- 定性：价值投资 / 成长投资 / 周期博弈 / 主题投机
```

## 模板二：高弹性小盘股机会

```text
请在 A 股中寻找 5 个高弹性小盘股机会，风险可接受，但不要推荐明显垃圾股或无基本面的公司。

条件：
1. 总市值小于人民币 [上限] 亿元。
2. 非 ST、非 *ST、非退市整理股。
3. 近 20 个交易日日均成交额大于 [金额] 万元。
4. 所属行业具备成长故事，如 AI 算力、机器人、半导体设备、创新药、固态电池、低空经济、新能源出海等。
5. 最近 6 个月存在明确催化剂：定增、重组、国资入主、股权激励、重大订单、产品量产、政策支持、临床进展等。
6. 尚未充分盈利可以接受，但必须说明现金储备、融资能力和亏损收窄趋势。

输出：
- 5 个标的
- 为什么可能暴涨：触发逻辑
- 当前市场忽视点
- 财务质量
- 最大风险：退市、监管、造假、质押、诉讼、流动性、融资压力等
- 风险等级：低 / 中 / 高 / 极高
- 判断：投机 / 投资 / 事件驱动博弈
```

## 模板三：低价未盈利事件股

```text
请调研当前 A 股股价低于 [价格] 元、总市值低于人民币 [上限] 亿元的小市值公司。不要把低价本身视为低估，重点关注事件驱动和风险排除。

重点特征：
1. 最近 6 个月内有重大资产重组、管理层大换血、国资入主、产业资本注资、定增落地、股权激励或重大订单。
2. 属于 AI、机器人、半导体、固态电池、创新药、低空经济、新能源等高增长赛道。
3. 目前尚未盈利或盈利较弱可以接受，但需要看到现金储备、融资能力、亏损收窄或订单兑现路径。

必须剔除：
ST、*ST、退市整理、暂停上市、近一年被立案调查、重大违法违规、审计意见非标、重大诉讼缠身、控股股东资金占用、成交极度低迷的公司。

请分析：
- 被市场低估或冷落的潜在原因
- 催化剂是否真实、是否有公告证据
- 未来 3-12 个月可能触发重估的节点
- 最大风险
- 风险等级评分：1-5 分，5 分最高风险
```

## 模板四：单家公司分析

```text
请分析这家 A 股公司：[公司名称]（[股票代码]）。

从以下维度：
1. 商业模式：怎么赚钱，收入来源是什么。
2. 行业地位：竞争对手是谁，优势和短板是什么。
3. 财务质量：收入、利润、扣非利润、经营现金流、负债、毛利率、应收账款、存货。
4. 估值是否合理：对比同行 PE、PB、PS、市值和成长性。
5. 未来增长逻辑：未来 6-24 个月看什么变量。
6. 潜在风险：政策、行业、技术、客户、财务、治理、交易风险。

最后给出：
- 多头逻辑：为什么可能涨。
- 空头逻辑：为什么可能跌。
- 当前更像：投机 / 投资 / 事件驱动博弈。
- 需要继续跟踪的 3 个公告或数据节点。
```
