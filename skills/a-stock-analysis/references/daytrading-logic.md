# 做T逻辑详解

## 什么是做T

A股 T+1 制度下，当天买入的股票次日才能卖出。但**当天已持有的股票可以卖出后再当天回补**，利用日内波动降低持仓成本，称为"做T"。

```
已持有 1000 股 @ 成本 85.5 元
→ 当日 90 元卖出 500 股（兑现盈利 +4.5×500 = 2250 元）
→ 当日 87 元回补 500 股（重新买回，少花 3×500 = 1500 元）
→ 综合持仓成本从 85.5 降至 ~84.75 元
```

---

## 做T两种策略

### 策略一：技术位做T（适合有波动的强势股）

| 操作 | 价格参考 | 来源 |
|------|---------|------|
| **卖出** | BOLL 上轨 / 日内前高 / 涨幅 > +4% | 压力位，大概率遇阻回落 |
| **回补** | MA5 / 当日开盘价 / 日内低点 + 0.5% | 支撑位，大概率企稳 |

**适用条件**：
- RSI 65～80（偏强但未严重超买）
- 当日振幅预计 > 3%
- BOLL 上下轨间距 > 5%

---

### 策略二：成本优化做T（适合震荡盘整期）

目标：用日内高低点差降低持仓成本，不求最大收益，求稳定降本。

**触发条件**：当前价 ≥ 成本 × 1.03（即浮盈 ≥ 3%）
**回补目标**：成本 × 1.015 以下（比成本高 1.5%，保留一定安全垫）
**成本改善**：卖出比例 × (卖出价 - 回补价) / 总持仓成本

**示例**：
```
成本：85.5 元，持有 1000 股
卖出触发：≥ 85.5 × 1.03 = 88.07 元时卖出 300 股
回补目标：≤ 85.5 × 1.015 = 86.78 元回补
成本降低：(88.07 - 86.78) × 300 / (85.5 × 1000) ≈ 0.45%
```

## 做T适宜度评分

| 因素 | 高适宜（+） | 低适宜（-） |
|------|-----------|-----------|
| 振幅 | 当日振幅 > 3% | 振幅 < 1.5%（横盘） |
| 资金 | 超大单活跃，方向多变 | 超大单单向，趋势强 |
| RSI | 65～80（做空间） | > 80 或 < 40 |
| BOLL | 上下轨间距宽 | 布林带收窄（即将变向） |
| 时段 | 10:00～11:30 / 13:30～14:30 | 开盘 9:30～9:45（波动大风险高） |

---

## 不建议做T的情形

- **趋势太强**：连续放量上涨，卖出后可能追不回来
- **消息面不确定**：有重大公告未消化时，不做T
- **仓位太小**：持仓金额 < 2 万元，做T收益不覆盖手续费
- **RSI > 80**：超买严重，可能直接回调，应该全面减仓而非做T

## 止损位计算规则

```
止损位 = max(成本 × 0.95, MA20)
```

- 成本 × 0.95：最大亏损容忍 5%
- MA20：均线破位止损
- 取**较高者**：宁可止损早一点，不承担均线破位的额外风险

**示例**：
```
沪电股份成本 85.5，MA20 = 81.94
成本×0.95 = 81.23，MA20 = 81.94
止损位 = max(81.23, 81.94) = 81.94 元（以MA20为准）
```

---

## 止盈位计算规则

```
止盈位 = min(BOLL上轨, 成本 × 1.15)
```

- BOLL 上轨：技术压力目标
- 成本 × 1.15：+15% 收益目标
- 取**较低者**：先到先止盈，不贪高

**示例**：
```
沪电股份成本 85.5，BOLL上轨 = 93.55
成本×1.15 = 98.33
止盈位 = min(93.55, 98.33) = 93.55 元（以BOLL上轨为准）
```

---

## 风险声明

以上策略均为辅助参考，不构成投资建议。实际操作须结合当日消息面、大盘情绪、个人资金情况综合判断。
