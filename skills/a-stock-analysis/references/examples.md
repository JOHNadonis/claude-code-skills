# 使用示例

## 示例 1：单只股票买点分析

**用户输入**：「帮我分析一下宁德时代（300750）的买点」

```bash
# Step 1：行情 + BOLL + 量比
cd ~/.claude/skills/MX_FinData && python3 scripts/get_data.py \
  --query "宁德时代300750 最新股价 量比 BOLL布林线 换手率"

# Step 2：均线 + RSI + 成交量趋势
cd ~/.claude/skills/MX_FinData && python3 scripts/get_data.py \
  --query "宁德时代300750 RSI 5日均线 10日均线 20日均线 近期收盘价 5日均量"

# Step 3：资金 + 风险标记
cd ~/.claude/skills/MX_FinData && python3 scripts/get_data.py \
  --query "宁德时代300750 主力净流入 超大单净流入 解禁日期 大股东质押比例"
```

**量化过滤（输出前先判断）：**
- 量比 1.8 → 正常活跃，技术信号可信
- 量比 0.6 → 缩量，BOLL/RSI信号降级，不追

## 示例 3：做T完整流程

```bash
# 场景：持有 沪电股份(002463) 1000股 @ 85.50元
# 今日高点 90.00，回补目标 87.00

# 执行做T（自动计算新成本）
python3 scripts/portfolio.py t0 002463 300 90.00 87.00

# 输出：
# ✅ 做T完成：沪电股份(002463)
#    卖出：300股 @ 90.00元
#    回补：300股 @ 87.00元
#    做T收益：+900.00 元
#    成本从 85.50 → 84.60 元（降低 0.90 元）
```

## 数据解读速查

### 量比快速判断（量化时代最重要的过滤器）
| 量比 | 信号质量 | 建议 |
|------|---------|------|
| > 3.0 | 极强异动 | 确认方向，警惕量化诱多/诱空 |
| 2.0～3.0 | 强势 | 技术信号高度可信，可积极操作 |
| 1.0～2.0 | 正常 | 技术信号正常参考 |
| < 0.8 | 缩量 | 技术信号降级，不建议做T |
| < 0.5 | 地量 | 横盘待变，等放量确认方向 |

### 竞价形态快速判断
| 竞价形态 | 含义 | 操作意义 |
|---------|------|---------|
| 高开 + 量比>2 | 真启动，量化确认 | 可积极参与 |
| 高开 + 量比<0.8 | 假高开/洗盘 | 等回落再介入 |
| 低开 + 放量 | 恐慌盘 | 观察企稳，不急于止损 |
| 平开 + 缩量 | 方向未定 | 等9:45后信号明确 |

### RSI 快速判断
| RSI | 含义 | 操作 |
|-----|------|------|
| < 30 | 超卖 | 关注反弹，分批买入 |
| 30～50 | 弱势震荡 | 观望为主 |
| 50～70 | 健康上升 | 可介入 |
| > 70 | 超买 | 不追高，等回调 |
| > 80 | 严重超买 | 规避 |

### 资金信号速查
| 信号组合 | 含义 |
|---------|------|
| 超大单净流入 + 小单净流出 | 机构建仓，最强买入信号 |
| 超大单净流出 + 小单净流入 | 机构出货散户接盘，危险 |
| 连续3日主力净流出 | 主力离场，谨慎 |
| 首次转正（之前持续流出）| 主力开始布局，轻仓观察 |

### 龙虎榜信号速查
| 龙虎榜形态 | 含义 |
|---------|------|
| 机构席位净买入 > 游资 | 最强信号，确定性高 |
| 游资连续2日净买入 | 短线机会，关注出货节奏 |
| 机构卖出 + 游资买入 | 接力游戏，风险偏高 |
| 全是游资，无机构 | 纯题材炒作，把握好退出时机 |

### BOLL 布林线快速判断
| 位置 | 含义 |
|------|------|
| 股价 > 上轨 | 超强势，超买风险高 |
| 上轨～中轨 | 多头区间，正常运行 |
| 股价 = 中轨 | 趋势中性，方向待确认 |
| 中轨～下轨 | 空头区间，谨慎 |
| 股价 < 下轨 | 超卖，可能反弹 |
