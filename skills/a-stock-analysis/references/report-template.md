# 操作决策报告模板

> 此文件由 `intraday.py` 的 prompt 引用，Claude 生成报告时严格按此格式输出。

---

## 一、消息面总结（必须排第一，这是所有操作的前提）

| 类型 | 内容 | 对持仓影响 |
|------|------|----------|
| 宏观（美股/期货/政策） | ... | 利好 / 利空 / 中性 |
| 板块动态 | ... | 利好 / 利空 / 中性 |
| 个股公告 | 来自MX公告数据 | 利好 / 利空 / 中性 |

**消息面综合定性**：偏多 / 中性 / 偏空
**对今日操作的影响**：一句话说明消息面是否支撑技术操作

---

## 二、竞价信号解读（量化时代新增，排第二）

> 集合竞价是量化系统的"意图披露"时刻，先于所有技术面判断。

| 股票 | 竞价价格 | vs昨收 | 竞价量比 | 形态判断 | 操作含义 |
|------|---------|--------|---------|---------|---------|
| XX(代码) | XX.XX | +X.X% | X.X | 高开放量/高开缩量/低开放量/平开 | 真启动/洗盘/恐慌盘/方向待定 |

---

## 三、大盘判断

- 点位、涨跌方向、成交量（萎缩/放量）、北向资金方向
- 整体风险等级：**低 / 中 / 高**（消息面 + 技术面综合）

---

## 四、个股逐一分析

每只股票输出以下格式：

```
【股票名（代码）】
持仓成本：X.XX 元 | 当前价：X.XX 元 | 浮盈亏：+X.X% / 盈亏金额：+X,XXX 元

消息面：[来自公告/板块动态，无则写「无重大消息」]

量化信号（过滤器）：
  量比：X.X（高/正常/低）→ 技术信号可靠性：高/中/低
  竞价形态：[高开放量/高开缩量/低开放量/平开]
  龙虎榜：[机构净买入X亿 / 游资净买入X亿 / 未上榜]

技术信号：RSI=XX（超买/正常/超卖）| 均线：多头/空头 | BOLL位置：上轨/中轨/下轨附近
资金信号：超大单净流入 +X.XX亿 ✅/❌ | 主力净流入 +X.XX亿 ✅/❌

综合共振：消息×量化×技术×资金 → 共振方向：做多/做空/分歧
操作建议：持有 / 减仓 / 加仓 / 观望（1句理由，必须提到消息面和量比是否支撑）

【做T建议】
  适宜度：高/中/低（低时说明原因，量比<0.8 直接标低）
  技术位做T：卖出参考 XX.XX 元，回补参考 XX.XX 元，预计收益约 +X.X%
  成本优化做T：触发价 ≥ XX.XX 元（成本+3%），回补目标 ≤ XX.XX 元（成本+1.5%）
  执行命令：python3 scripts/portfolio.py t0 [代码] [股数] [卖出价] [回补价]
  ⚠️ 消息面风险：[若有公告/重大事项，说明是否影响做T安全性]

【风险标记】
  解禁风险：距解禁日 XX 天（< 30天则收紧止损）
  质押风险：大股东质押比例 XX%（> 40% 则不建议加仓）
  融资变化：融资余额 +/-X%（持续增加=信心，持续减少=出逃）

【止损位】XX.XX 元（= max(MA20, 成本×0.95)，若距解禁<30天则用成本×0.97）
【止盈位】XX.XX 元（= min(BOLL上轨, 成本×1.15)）
【持有评级】★★★☆☆（消息/量化/技术/资金四维综合，1-5星）
  - 质押>40% 或距解禁<30天：强制降一星
```

## 六、潜在机会提示（可选）

若大盘/板块出现明显异动，或有持仓之外的短线机会（龙虎榜新上榜、竞价异动），简要提示。

## 七、风险提示

- 今日最大外部风险（消息面）
- 个股特有风险（解禁/质押/公告）
- 今日重点监控价位清单

## 操作记录命令速查

```bash
# 做T后自动降成本
python3 scripts/portfolio.py t0 [代码] [卖出股数] [卖出价] [回补价]

# 加仓（自动加权平均成本）
python3 scripts/portfolio.py add [代码] [名称] [股数] [价格]

# 部分止盈
python3 scripts/portfolio.py sell [代码] [股数] [价格]

# 查复盘记录
python3 scripts/portfolio.py history [代码]
```
