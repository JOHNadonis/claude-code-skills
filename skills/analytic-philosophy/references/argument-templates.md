# 论证模板库

分析哲学中高频出现的论证形式，用于论证重构和批判评估。

---

## 一、基础论证形式

### 1. 演绎有效形式

**肯定前件 (Modus Ponens)**
```
P1: 如果 A，则 B
P2: A
∴ C: B
```

**否定后件 (Modus Tollens)**
```
P1: 如果 A，则 B
P2: 非 B
∴ C: 非 A
```

**假言三段论**
```
P1: 如果 A，则 B
P2: 如果 B，则 C
∴ C: 如果 A，则 C
```

---

### 2. 归谬法 (Reductio ad Absurdum)

```
假设 H（我们想推翻的命题）
P1: H 蕴含 X
P2: X 是荒谬的/矛盾的/不可接受的
∴ 非 H
```

**使用场景**：当你无法直接证明某命题为假时，证明其导致矛盾。

**分析哲学经典用例**：
- 罗素推翻朴素集合论（罗素悖论）
- 摩尔批自然主义（自然主义谬误论证）

---

### 3. 最佳解释推论 (Inference to the Best Explanation)

```
P1: 现象 E 存在
P2: 理论 H 能最好地解释 E
P3: 没有竞争理论比 H 更好地解释 E
∴ H 可能为真
```

**批判要点**：
- "最好解释"的标准是什么？
- 是否还有未考虑的竞争解释？

---

### 4. 概念分析论证

```
P1: X 的定义是：满足条件 C1, C2, C3 的东西
P2: [反例] 存在某个 Y，满足 C1, C2, C3 但直觉上不是 X
∴ 该定义不充分（或不必要）
```

**使用场景**：哲学家提出 X 的定义时，用反例攻击。

**经典实例**：盖梯尔用反例攻击知识的 JTB 定义。

---

## 二、哲学专用论证模式

### 5. 可能性论证 (Possibility Argument)

```
P1: S 是可以设想的 (conceivable)
P2: 如果 S 可设想，则 S 是可能的
P3: 如果 S 是可能的，则命题 P 为真
∴ P 为真
```

**批判要点**：
- 可设想性真的蕴含可能性吗？（查默斯 vs 批评者）
- P1 是否真的可设想？

**经典实例**：哲学僵尸论证（查默斯）

---

### 6. 思想实验 + 直觉泵

```
场景构造：[设计一个与日常接近但关键变量不同的情境]
直觉抽取：在这个情境中，你的判断是什么？
外推：如果直觉是 J，则 J 支持/推翻原则 P
```

**使用注意**：
- 思想实验测试的是直觉，直觉可能有偏差
- 需要解释为什么这个直觉比对立直觉更可靠

---

### 7. 两难困境论证 (Dilemma)

```
P1: 要么 A，要么 B（穷举）
P2: 如果 A，则 C（不可接受）
P3: 如果 B，则 D（不可接受）
∴ 无论 A 还是 B，结果都不可接受
∴ 原始假设/框架有问题
```

**批判要点**：
- P1 真的穷举了所有可能吗？（抓角）
- 能否接受 C 或 D（咬子弹）？

---

### 8. 叠加论证 (Sorites / 连锁悖论)

```
P1: 一粒沙不是一堆沙
P2: 如果 n 粒沙不是一堆，则 n+1 粒也不是
∴ 任意多粒沙都不是一堆沙（显然荒谬）
```

**哲学意义**：揭示模糊概念的边界问题。
**回应策略**：精确主义 / 上限语义 / 容忍主义

## 三、批判评估框架

### 攻击一个论证的三条路

**路径 1：攻击前提（最常见）**
- 选择最脆弱的前提
- 构造反例或提供对立直觉
- 说明为什么这个前提不应被接受

**路径 2：攻击推论（形式有效性）**
- 指出推论形式无效（P1+P2 不能推出 C）
- 指出有缺失的中间步骤
- 指出论证依赖于未明说的额外假设

**路径 3：咬子弹（接受结论）**
- 承认论证有效，接受看似荒谬的结论
- 解释为什么结论其实没那么荒谬
- 修正产生"荒谬"感的直觉

---

### 辩护一个论证的三条路

**路径 1：强化最脆弱前提**
- 提供额外支持
- 回应反例（说明反例不适用或被误解）
- 区分两种不同的理解，反例只适用于其中一种

**路径 2：重构论证**
- 用更强的形式重新表述
- 替换脆弱前提为更可接受的版本
- 明确隐含前提，检验其可接受性

**路径 3：反攻**
- 对攻击者的论证提出同等力度的反例
- 指出对立立场有更严重的问题

## 四、常见谬误速查

| 谬误名称 | 描述 | 识别标志 |
|---------|------|---------|
| 稻草人谬误 | 攻击一个弱化/歪曲的对立观点 | "你的意思是……所以……" |
| 循环论证 | 前提预设了结论 | 前提和结论说的是同一件事 |
| 诉诸无知 | 因为无法证伪，所以为真 | "你证明不了它是假的" |
| 滑坡谬误 | 夸大因果链条而无支撑 | "A→B→C→……→灾难" |
| 自然主义谬误 | 从"是什么"推出"应该是什么" | "这是自然的，所以是好的" |
| 诉诸直觉的滥用 | 直觉有争议却被当作公理 | "任何人都会同意……" |
| 混淆使用/提及 | 说的是词本身还是词所指的对象 | "猫"有三个字母 vs. 猫有四条腿 |
