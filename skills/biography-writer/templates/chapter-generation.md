# 章节生成提示词模板

## 基础版本

```
请为第[X]章撰写内容。

【章节信息】
- 标题：{{CHAPTER_TITLE}}
- 时间跨度：{{START_YEAR}} - {{END_YEAR}}
- 核心事件：{{CORE_EVENTS}}
- 字数要求：{{WORD_COUNT}}字

【人物】
{{CHARACTER_NAME}}

【写作要求】
严格遵循项目 CLAUDE.md 中的三层结构法：
1. 叙事层：场景化呈现核心事件
2. 人物层：挖掘动机与人际关系
3. 制度层：交代时代背景约束

【质量要点】
- 开篇500字内必须有钩子（悬念/冲突/意外）
- 重要史实标注来源："据《XX》记载..."
- 对不确定处标注：【存疑】【待补充来源】【争议点】
- 避免：上帝视角、后见之明、AI典型句式

完成后保存到 chapters/{{CHAPTER_NUMBER}}-{{CHAPTER_SLUG}}.md
```

---

## 并行生成版本

```
请同时启动 {{N}} 个写作任务，并行撰写以下章节：

【任务列表】
{{#each CHAPTERS}}
任务{{@index}}：
- 章节：第{{this.number}}章 - {{this.title}}
- 时间：{{this.start_year}} - {{this.end_year}}
- 核心事件：{{this.events}}
- 字数：{{this.word_count}}字
- 输出：chapters/{{this.number}}-{{this.slug}}.md
{{/each}}

【共同要求】
- 人物：{{CHARACTER_NAME}}
- 严格遵循项目 CLAUDE.md 中的三层结构法
- 每章开篇必须有钩子
- 重要史实标注来源
- 自我标注不确定处

【质量红线】
- ❌ 禁止上帝视角
- ❌ 禁止后见之明
- ❌ 禁止 AI 典型句式
- ❌ 禁止形容词堆砌
```

## 特定章节类型模板

### 开篇章（出身与早年）

```
请撰写传记开篇章：{{CHARACTER_NAME}}的出身与早年。

【时间跨度】{{BIRTH_YEAR}} - {{EARLY_YEAR}}

【重点呈现】
1. 家世背景：家族渊源、社会地位、经济状况
2. 成长环境：地域特点、时代氛围、教育经历
3. 性格形成：关键事件、重要影响、早期特质
4. 伏笔埋设：为后续发展埋下线索

【开篇技巧】
- 可从一个具体场景切入
- 可从一个命运转折点倒叙
- 可从一个核心矛盾引出

【字数】{{WORD_COUNT}}字
【输出】chapters/01-origin.md
```

### 转折章（关键决策）

```
请撰写转折章：{{CHARACTER_NAME}}在{{EVENT_NAME}}中的抉择。

【时间跨度】{{START_YEAR}} - {{END_YEAR}}

【核心问题】
- {{CHARACTER_NAME}}面临什么困境？
- 有哪些选项？各有什么代价？
- 最终为什么选择了这条路？
- 这个选择如何改变了后续轨迹？

【理论视角】（选用1-2个）
- 博弈论：多方互动分析
- 委托代理：信息不对称分析
- 路径依赖：为何不走"更好的路"

【叙事技巧】
- 营造决策前的紧张感
- 呈现不同声音的交锋
- 避免事后诸葛式评判

【字数】{{WORD_COUNT}}字
【输出】chapters/{{CHAPTER_NUMBER}}-{{CHAPTER_SLUG}}.md
```

### 高潮章（巅峰时刻）

```
请撰写高潮章：{{CHARACTER_NAME}}的{{PEAK_EVENT}}。

【时间跨度】{{START_YEAR}} - {{END_YEAR}}

【重点呈现】
1. 事件经过：完整还原关键过程
2. 人物表现：在压力下的真实反应
3. 多方视角：不同参与者的立场
4. 历史意义：对时代的影响（克制表述，不要拔高）

【场景化要求】
- 至少3个具体场景详细展开
- 有具体的时间、地点、人物、对话
- 让读者"身临其境"

【字数】{{WORD_COUNT}}字（本章可适当超出）
【输出】chapters/{{CHAPTER_NUMBER}}-{{CHAPTER_SLUG}}.md
```

### 结尾章（晚年与评价）

```
请撰写结尾章：{{CHARACTER_NAME}}的晚年与身后。

【时间跨度】{{LATE_YEAR}} - {{DEATH_YEAR}}（及身后）

【重点呈现】
1. 晚年生活：最后阶段的状态与心境
2. 临终场景：如有记载，详细呈现
3. 身后评价：同时代人的评价（多元视角）
4. 历史回响：后世如何看待（呈现演变，不下定论）

【写作原则】
- 不要"盖棺定论"式的总结
- 呈现争议而非裁决争议
- 与开篇形成呼应
- 留给读者思考空间

【字数】{{WORD_COUNT}}字
【输出】chapters/{{CHAPTER_NUMBER}}-ending.md
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{CHARACTER_NAME}}` | 传主姓名 | 李鸿章 |
| `{{CHAPTER_NUMBER}}` | 章节序号 | 01, 02, 03... |
| `{{CHAPTER_TITLE}}` | 章节标题 | 科举与初仕 |
| `{{CHAPTER_SLUG}}` | 章节简称（英文） | early-career |
| `{{START_YEAR}}` | 起始年份 | 1843 |
| `{{END_YEAR}}` | 结束年份 | 1853 |
| `{{CORE_EVENTS}}` | 核心事件列表 | 考中进士、入翰林院... |
| `{{WORD_COUNT}}` | 目标字数 | 10000 |
| `{{N}}` | 并行任务数 | 3 |
