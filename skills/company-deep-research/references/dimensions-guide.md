# 七维分析模型详解

> 每个维度的评分标准、信号识别、红绿灯指标。

---

## 维度一：管理层决策质量

**核心问题：** 这个人在关键分叉口的判断力如何？

### 评分标准

| 分数 | 标准 |
|------|------|
| 5分 | 有3次以上被市场验证正确的非共识判断 |
| 4分 | 有明确的战略一致性，关键决策有正向结果 |
| 3分 | 管理稳定，无明显失误，但也无突出判断力 |
| 2分 | 有过重大战略失误，或频繁更换方向 |
| 1分 | 管理层频繁变动，或有诚信问题 |

### 绿灯信号
- 创始人在逆境中拒绝低价收购（说明知道价值被低估）
- 主动降薪/零薪+增持（把身家绑在公司上）
- 10年以上专注同一赛道（时间一致性）
- 公开演讲的预判后来被验证

### 红灯信号
- 高管连续减持，尤其是创始人减持
- 频繁跨行业收购（什么热做什么）
- 实控人频繁质押股权
- 管理层年薪 vs 公司利润比例畸高

### 信息来源
- Exa people_search → 创始人公开信息
- WebSearch → 采访、演讲、创业故事
- MX_FinData → 高管持股变动、股权激励

## 维度二：市场结构裂缝

**核心问题：** 大厂"觉得没必要做"的地方在哪？这家公司是在挤破头的红海，还是在别人不走的路上？

### 评分标准

| 分数 | 标准 |
|------|------|
| 5分 | 开创了新品类/新路径，竞争对手尚未跟进 |
| 4分 | 在大厂忽视的细分市场建立了明确优势 |
| 3分 | 跟随行业主流路线，有一定差异化 |
| 2分 | 纯跟随者，无差异化，靠价格竞争 |
| 1分 | 红海竞争，无护城河，替代性强 |

### 判断方法
1. 找出行业前三名的战略重心在哪
2. 看这家公司做的事和前三名是否重叠
3. 重叠度低 + 市场真实存在 = 裂缝
4. 重叠度高 = 红海

### 经典模式
- "大厂嫌量小不做" → 小公司做 → 量起来后大厂来不及了
- "大厂不愿定制" → 小公司做定制 → 客户锁定
- "大厂只做高端" → 小公司做中低端 → 逐步上攻

## 维度三：技术周期位置

**核心问题：** 这个技术从实验室到大规模商业化，走到哪一步了？

### 评分标准

| 分数 | 标准 |
|------|------|
| 5分 | 技术领先且即将进入量产放量期（最佳入场点）|
| 4分 | 技术成熟，已量产，市场渗透率快速提升 |
| 3分 | 技术可用，但市场接受度还在早期 |
| 2分 | 技术尚未成熟，量产时间不确定 |
| 1分 | 技术路线存疑，可能被替代 |

### 技术生命周期判断

```
实验室 → 小试 → 中试 → 量产 → 放量 → 成熟 → 衰退
                        ↑
                    最佳投资窗口
```

### 关键信号
- 专利申请数量加速（提前1-3年的信号）
- 从"研发投入"转向"资本开支"（说明准备量产）
- 下游客户开始小批量验证
- 竞争对手开始跟进（说明路线被验证）

### 信息来源
- WebSearch → 专利数据库、技术文章
- WebSearch → 行业会议报告
- MX_FinData → 研发费用/资本开支趋势

## 维度四：产业链位置

**核心问题：** 公司在价值链的哪一层？议价权在哪？

### 评分标准

| 分数 | 标准 |
|------|------|
| 5分 | 处于价值链关键节点，上下游都依赖它，替代成本高 |
| 4分 | 有明确的技术/渠道壁垒，客户切换成本高 |
| 3分 | 行业中游，有一定话语权但可被替代 |
| 2分 | 纯代工/纯渠道，议价权弱 |
| 1分 | 产业链最底端，完全被动接受定价 |

### "被忽略的中间层"识别

最值钱的公司往往不在产业链最显眼的位置：
- 上游（原材料/IP）：被看到，被定价
- 下游（品牌/终端）：被看到，被定价
- **中间层（方案设计/模组/集成）**：往往市值最小，但利润密度最高

### 判断方法
1. 画出3层产业链图谱：上游→中间→下游
2. 标出每层的代表公司和毛利率
3. 毛利率最高的那层 = 议价权所在
4. 如果目标公司在毛利率最高的层且市值最小 = 低估

### 信息来源
- MX_FinData → 毛利率对比、客户集中度
- WebSearch → 产业链图谱、上下游分析
- 客户/供应商年报 → 交叉验证议价权

## 维度五：资本结构

**核心问题：** 谁在出钱？结构是否抗风险？

### 评分标准

| 分数 | 标准 |
|------|------|
| 5分 | 现金流自给自足，股东结构稳定，无质押风险 |
| 4分 | 有战略投资者背书，资金充裕 |
| 3分 | 正常融资，资产负债率行业平均 |
| 2分 | 高负债经营，质押比例较高 |
| 1分 | 资金链紧张，质押比例 > 40%，有爆仓风险 |

### 关键指标
- 资产负债率 vs 行业平均
- 质押比例（> 40% 强红灯）
- 经营现金流 / 净利润（< 0.7 连续2年 = 盈利质量差）
- 股东结构变化趋势（国资/产业资本进入 = 正向信号）

## 维度六：隐含信号

**核心问题：** 公开数据之外，什么信号在提前暗示方向？

### 三大隐含信号源

**招聘信号（领先公告 6-12 个月）**
- 大量招AI算法工程师 → 可能在布局AI产品线
- 招海外销售团队 → 准备出海
- 招合规/法务 → 可能在准备上市或应对监管
- 核心岗位停招 → 业务可能收缩

**子公司/工商变更（领先公告 3-6 个月）**
- 新设子公司的名称和经营范围 → 新业务方向
- 注册资本变更 → 加注还是收缩
- 对外投资/参股 → 产业链延伸方向

**专利时间线（领先产品 1-3 年）**
- 专利申请数量突然加速 → 技术突破可能临近
- 专利方向突然转变 → 技术路线调整
- 海外专利布局 → 全球化准备

## 维度七：风险矩阵 + 反面论证

**核心问题：** 如果我的判断完全错了，最可能错在哪里？

### 标准风险清单（逐项检查）

| 风险类型 | 检查项 |
|---------|--------|
| 解禁风险 | 未来6个月内有大额解禁？ |
| 质押风险 | 实控人质押比 > 30%？ |
| 诉讼风险 | 有未了结的重大诉讼？ |
| 政策风险 | 行业是否面临监管收紧？ |
| 技术替代 | 是否有替代技术路线在崛起？ |
| 客户集中 | 前3大客户占收入 > 60%？ |
| 周期风险 | 行业是否处于周期顶部？ |
| 估值风险 | 当前PE是否在历史90%分位以上？ |

### 反面论证模板

```
我的核心投资论点是：[一句话]

这个论点失效的三种情况：
1. [技术/市场维度] 如果...那么...
2. [竞争/政策维度] 如果...那么...
3. [管理/资本维度] 如果...那么...

各情况的发生概率评估：高/中/低
综合论点置信度：高/中/低
```
