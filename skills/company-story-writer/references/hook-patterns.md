# 开头钩子句式库

本文档提供公众号文章开头的句式模板和钩子模式，帮助快速构建吸引读者的开场。

---

## 核心原则

好的开头必须在前 3 段做到以下至少一点：
1. **制造反差** - 让读者产生"咦？"的感觉
2. **抛出判断** - 给出一个明确的观点
3. **设置悬念** - 让读者想知道"然后呢"

---

## 句式模板库

### 类型 1：认知反转型

**模板：**
```
很多人以为 [公司/人物] 是 [A]，其实它/他是 [B]。
```

**示例：**
- 很多人以为这家公司赚的是设备的钱，其实它真正值钱的是背后的数据网络。
- 很多人以为他是工程师出身，其实他是学哲学的。
- 很多人以为这是一家资源股，其实它是一家技术公司。

**变体：**
```
如果只看 [表面现象]，你很难发现 [真相]。
```

**示例：**
- 如果只看财报，你很难发现这家公司真正特别的地方。
- 如果只看产品，你不会觉得它和同行有什么不同。

---

### 类型 2：对比冲击型

**模板：**
```
[时间点]，当所有人都在做 [A] 的时候，他/它在做 [B]。
```

**示例：**
- 2008 年，当所有光伏企业都在拼命扩产能的时候，他做了一个在当时看起来很蠢的决定：停止扩张，把钱投到技术研发上。
- 2015 年，当所有新能源车企都在拼续航的时候，这家公司选择了换电。
- 去年，当所有人都在降价抢市场的时候，它反而涨价了。

**变体：**
```
别人在 [做A]，它在 [做B]。结果是：[反转结果]。
```

**示例：**
- 别人在拼规模，它在拼效率。结果是：别人的毛利率从 20% 跌到 5%，它的毛利率从 15% 涨到 18%。

---

### 类型 3：数据冲击型

**模板：**
```
[数据1] vs [数据2]，这个反差说明了什么？
```

**示例：**
- 市值 300 亿，但账上现金 200 亿。这个反差说明了什么？
- 营收只有行业老大的十分之一，但利润率是行业老大的两倍。
- 员工只有 500 人，但人均创收 1000 万。

**变体：**
```
[时间段] 内，[主体] 的 [指标] 从 [A] 变成了 [B]。
```

**示例：**
- 三年内，这家公司的毛利率从 12% 涨到 35%。
- 五年前，它的市值只有 30 亿。今天，300 亿。

---

### 类型 4：他人评价型

**模板：**
```
[权威人物/机构] 说：[引用]。他/它说的是 [公司名]。
```

**示例：**
- 去年行业大会上，某龙头企业的 CTO 公开表示："我们最担心的不是谁的设备更便宜，而是谁先把数据网络建起来。" 他没点名，但所有人都知道他在说谁。
- 一位资深分析师在研报里写："这家公司最值钱的不是它的产品，是它的位置。"
- 它的竞争对手在财报电话会上说："我们不担心价格战，我们担心的是有人改变了游戏规则。"

---

### 类型 5：时间锚点型

**模板：**
```
[时间点]，[事件]。那一年，[背景]。今天，[结果]。
```

**示例：**
- 2010 年，他做了一个决定：放弃低价策略，做高效组件。那一年，他的公司营收只有行业老大的十分之一。今天，他是行业老大。
- 2015 年，这家公司差点被卖掉。那一年，它的市值只有 30 亿。今天，300 亿。

---

### 类型 6：问题开场型

**模板：**
```
为什么 [现象]？
```

**示例：**
- 为什么这家公司的毛利率能做到 35%，而同行只有 15%？
- 为什么所有人都在做 A，只有它在做 B？
- 为什么它能在红海市场里活得这么好？

**变体：**
```
[主体] 最值钱的不是 [A]，是 [B]。为什么？
```

---

### 类型 7：争议开场型

**模板：**
```
关于 [公司/人物]，市场分成了两派。一派说 [观点A]，另一派说 [观点B]。
```

**示例：**
- 关于这家公司，市场分成了两派。一派说它是"被低估的成长股"，另一派说它是"高估值的周期股"。谁对谁错？
- 最近这家公司的股价很有意思：一边是机构在加仓，一边是散户在抛售。

---

### 类型 8：故事开场型

**模板：**
```
[时间点]，[人物] 做了一个 [决定]。[背景/阻力]。[结果]。
```

**示例：**
- 2013 年，董事会开会讨论要不要卖掉公司。有人说"卖了吧，这个行业没前途了"。创始人说："再等一年。" 那一年，他赌对了。
- 2008 年，他去银行贷款，被拒了三次。第四次，他抵押了自己的房子。

## 钩子组合模式

### 模式 1：反差 + 数据

```
很多人以为 [A]，其实是 [B]。

[数据支撑]。

这个反差背后，是 [洞察]。
```

**示例：**
```
很多人以为这家公司赚的是设备的钱，其实它真正值钱的是背后的数据网络。

去年，它的设备销售收入 50 亿，但数据服务收入只有 5 亿。但如果你看毛利率，设备是 15%，数据服务是 60%。

这个反差背后，是一个很多人没看懂的商业模式。
```

---

### 模式 2：对比 + 结果

```
[时间点]，当所有人都在做 [A] 的时候，他/它在做 [B]。

[阻力/质疑]。

[结果]。
```

**示例：**
```
2010 年，当所有光伏企业都在拼命扩产能的时候，他做了一个在当时看起来很蠢的决定：停止扩张，把钱投到技术研发上。

董事会有人反对，说"市场要的是便宜，不是高效"。

2013 年，补贴开始退坡。那些靠低价抢市场的企业，利润率从 20% 跌到 5%。他的公司，利润率从 15% 涨到 18%。
```

---

### 模式 3：问题 + 反常识答案

```
为什么 [现象]？

大多数人的答案是 [常识答案]。

但真正的答案是 [反常识答案]。
```

**示例：**
```
为什么这家公司的毛利率能做到 35%，而同行只有 15%？

大多数人的答案是：技术更好、产品更高端。

但真正的答案是：它选择了一个别人不做的细分市场。
```

---

### 模式 4：他人评价 + 解读

```
[权威人物/机构] 说：[引用]。

这句话说的是 [公司名]。

为什么他/它这么说？
```

**示例：**
```
去年行业大会上，某龙头企业的 CTO 公开表示："我们最担心的不是谁的设备更便宜，而是谁先把数据网络建起来。"

他没点名，但所有人都知道他在说谁。

为什么他这么担心？因为一旦数据网络建起来，设备就不再是竞争的核心。
```

## 开头写作检查清单

写完开头后，检查以下几点：

- [ ] 前 3 段是否出现了反差、判断或悬念
- [ ] 是否用了具体的数据或事实（而非抽象描述）
- [ ] 是否让读者产生了"咦？"或"然后呢？"的感觉
- [ ] 是否避免了"深度分析""全面解读"这类死词
- [ ] 是否在前 8 段回答了"为什么今天值得讲"

## 常见错误

### 错误 1：开头太平淡

**不好的开头：**
```
XX 公司是一家专注于 XX 领域的企业，成立于 XX 年，主营业务包括 XX、XX 和 XX。
```

**为什么不好：**
- 没有任何钩子
- 读起来像公司简介
- 读者没有理由继续往下看

**改进：**
先抛出一个反差或判断，再补充背景信息。

---

### 错误 2：开头太抽象

**不好的开头：**
```
这家公司很厉害，在行业里有很强的竞争力，未来发展潜力巨大。
```

**为什么不好：**
- 全是空话
- 没有具体信息
- 读者不知道"厉害"在哪里

**改进：**
用具体的数据、事实、对比来说明"厉害"。

---

### 错误 3：开头太长

**不好的开头：**
```
（前 5 段都在铺垫背景，没有进入正题）
```

**为什么不好：**
- 读者没有耐心
- 钩子出现太晚
- 容易被划走

**改进：**
第 1 段就要有钩子，背景信息可以后面再补。

## 实战案例

### 案例 1：人物下注型文章开头

```
很多人以为他是工程师出身，其实他是学哲学的。

2008 年，当所有光伏企业都在拼命扩产能的时候，他做了一个在当时看起来很蠢的决定：停止扩张，把钱投到技术研发上。

那一年，他的公司营收只有行业老大的十分之一。今天，他是行业老大。

这个故事，值得讲。
```

**分析：**
- 第 1 段：反差（哲学 vs 工程师）
- 第 2 段：对比（所有人 vs 他）
- 第 3 段：结果（十分之一 vs 老大）
- 第 4 段：点题（为什么今天值得讲）

---

### 案例 2：公司命运型文章开头

```
2015 年，这家公司的市值只有 30 亿，在行业里排不进前十。

它的主营业务是传统的机械加工，毛利率 12%，净利率 3%，典型的"苦哈哈"制造业。

如果你在那一年看它的财报，你不会觉得它有什么特别。

但今天，它的市值是 300 亿。发生了什么？
```

**分析：**
- 第 1-3 段：铺垫"过去不重要"
- 第 4 段：抛出悬念（发生了什么）

---

### 案例 3：争议拆解型文章开头

```
最近这家公司的股价很有意思：一边是机构在加仓，一边是散户在抛售。

机构说它是"被低估的成长股"，散户说它是"高估值的周期股"。

谁对谁错？

这个问题的答案，比你想的更复杂。
```

**分析：**
- 第 1 段：现象（机构 vs 散户）
- 第 2 段：争议（成长股 vs 周期股）
- 第 3 段：问题
- 第 4 段：设置悬念

## 使用建议

1. **不要生搬硬套** - 这些模板是参考，不是公式。根据具体公司和文章角度灵活调整。

2. **先写正文，再优化开头** - 很多时候，写完正文后才能找到最好的开头角度。

3. **多写几个版本** - 每次至少写 3 个开头，选最好的那个。

4. **读出来听听** - 好的开头读起来应该很顺，不拗口。

5. **测试钩子强度** - 把开头发给朋友看，问他们"想不想继续往下读"。
