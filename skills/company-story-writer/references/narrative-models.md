# 五种叙事模型详解

本文档详细说明 company-story-writer 技能中的 5 种叙事模型。每种模型都有固定的结构和适用场景。

---

## 模型 A：人物下注型

### 适用场景
- 创始人/董事长有强烈的个人特质
- 公司的发展与创始人的决策密切相关
- 存在明显的"关键下注"时刻
- 适合写成"他为什么能看见别人看不见的"

### 文章结构

#### 1. 开头：反差钩子（200-300字）
用一个反差或意外的事实开场，让读者产生"咦？"的感觉。

**常见反差类型：**
- 背景反差：文科生做硬科技、学者做重工业
- 决策反差：所有人都在做 A，他在做 B
- 结果反差：当年不被看好的决策，今天成了护城河

**示例开头：**
```
很多人以为他是工程师出身，其实他是学哲学的。

2008 年，当所有光伏企业都在拼命扩产能的时候，他做了一个在当时看起来很蠢的决定：停止扩张，把钱投到技术研发上。

那一年，他的公司营收只有行业老大的十分之一。今天，他是行业老大。
```

#### 2. 人物背景（300-500字）
不是流水账式的履历，而是提炼出"这个人为什么会做出后来的决策"。

**要回答的问题：**
- 他的教育背景/职业经历中，哪些经历塑造了他的思维方式？
- 他进入这个行业的契机是什么？
- 他和同行最大的不同是什么？

**写作要点：**
- 不要罗列所有履历，只讲关键的 2-3 个节点
- 每个节点都要说明"这段经历给了他什么"
- 用具体细节而非抽象描述

#### 3. 第一次关键下注（400-600字）
公司/创始人的第一个重大决策。

**结构：**
- **背景**：当时的行业环境、主流做法
- **决策**：他做了什么不一样的选择
- **阻力**：这个决策遇到了什么反对/质疑
- **结果**：短期和长期的结果
- **反思**：为什么他敢这么做

**示例：**
```
2010 年，光伏行业的主流打法是"大规模、低成本"。所有人都在拼谁的产能更大，谁的价格更低。

他选择了另一条路：做高效组件。

这个决策在当时看起来很不明智。高效组件的成本更高，价格也更贵，在一个拼价格的市场里，这意味着没有竞争力。

董事会有人反对，说"市场要的是便宜，不是高效"。

他的回答是："今天市场要便宜，是因为补贴够多。等补贴退坡，市场要的就是高效。"

2013 年，补贴开始退坡。那些靠低价抢市场的企业，利润率从 20% 跌到 5%。他的公司，利润率从 15% 涨到 18%。
```

#### 4. 第二次关键下注（400-600字）
结构同上，但要体现"递进"关系——第二次下注是基于第一次的经验/教训。

#### 5. 第三次关键下注（400-600字）
如果有第三次，继续写。如果没有，可以省略这一部分。

#### 6. 为什么别人没看见，他看见了（300-500字）
这是文章的"升华"部分，要回答：

- 他的判断力来自哪里？
- 他和同行的思维方式有什么本质不同？
- 他的成功是运气还是能力？

**写作要点：**
- 不要写成"他很聪明""他很有远见"这种空话
- 要具体到思维模式、信息来源、决策框架
- 可以用对比：别人怎么想的，他怎么想的

#### 7. 这家公司值不值得继续看（200-300字）
结尾，给出明确判断。

**要回答的问题：**
- 这家公司今天处于什么位置？
- 它最大的变量是什么？
- 市场可能错在哪里？

**写作要点：**
- 不要模棱两可
- 不要只说好话，也要说风险
- 给读者一个可以行动的判断

## 模型 B：公司命运型

### 适用场景
- 老公司重生、国企转型、借壳翻身
- 公司的价值不在于它一直做什么，而在于它"变成了什么"
- 存在明显的"命运转折点"
- 适合写成"这家公司原来不重要，现在很重要"

### 文章结构

#### 1. 这家公司原来根本不重要（200-300字）
开场就说明：这家公司在过去很长一段时间里，是个"nobody"。

**要点：**
- 用具体数据说明"不重要"：市值、营收、行业排名
- 说明它当时在做什么（平淡无奇的业务）
- 为后面的"转折"做铺垫

**示例：**
```
2015 年，这家公司的市值只有 30 亿，在行业里排不进前十。

它的主营业务是传统的机械加工，毛利率 12%，净利率 3%，典型的"苦哈哈"制造业。

如果你在那一年看它的财报，你不会觉得它有什么特别。
```

#### 2. 它如何拿到关键资产/关键位置（400-600字）
转折点：公司通过某种方式（并购、重组、技术突破、政策红利等）拿到了"改变命运"的东西。

**要回答的问题：**
- 它拿到了什么？（资产、牌照、技术、客户、位置）
- 怎么拿到的？（并购、重组、自研、运气）
- 为什么是它拿到，而不是别人？

**写作要点：**
- 讲清楚"拿到"的过程，不要一笔带过
- 如果有戏剧性（差点失败、竞争激烈等），要写出来
- 说明这个资产/位置为什么重要

#### 3. 它什么时候开始变了（400-600字）
拿到关键资产后，公司的变化。

**要点：**
- 财务数据的变化（营收、利润、毛利率）
- 业务结构的变化（从做 A 变成做 B）
- 市场地位的变化（从行业边缘到核心）

**写作要点：**
- 用对比突出变化：过去 vs 现在
- 用具体数据，不要只说"变好了"
- 说明这个变化是一次性的还是可持续的

#### 4. 它现在真正值钱的不是表面业务（300-500字）
这是文章的"洞察"部分：市场可能还在用老眼光看这家公司，但它真正的价值已经变了。

**要回答的问题：**
- 市场以为它值钱的是什么？
- 它真正值钱的是什么？
- 为什么市场还没意识到？

**示例：**
```
很多人以为这家公司值钱的是它的设备制造能力，其实它真正值钱的是它在行业里的"位置"。

它现在是行业里唯一一家同时拥有 A 牌照和 B 资质的公司。这意味着，所有想做这个业务的企业，都得找它合作。

这个位置，不是靠技术能力拿到的，是靠历史机缘。而且，这个位置有很强的先发优势——后来者很难再拿到同样的牌照。
```

#### 5. 风险在哪里（200-300字）
不要只说好话，要说清楚风险。

**常见风险：**
- 关键资产的持续性（牌照会不会被取消、技术会不会被超越）
- 管理层的能力（拿到资产是运气，用好资产是能力）
- 行业环境的变化（政策、竞争、需求）

#### 6. 为什么今天要重估它（200-300字）
结尾，给出判断。

**要回答的问题：**
- 市场对这家公司的定价是否合理？
- 如果要重估，重估的逻辑是什么？
- 什么时候是好的介入时点？

## 模型 C：行业裂缝型

### 适用场景
- 公司走的是行业"偏门"、非主流路线
- 主流玩家都在做 A，它在做 B
- 适合写成"别人不做的地方，它在做"

### 文章结构

#### 1. 行业主流叙事是什么（200-300字）
开场先说明：这个行业的"正统打法"是什么。

**要点：**
- 主流玩家在做什么
- 主流路线的逻辑是什么
- 市场的共识是什么

**示例：**
```
在新能源汽车行业，主流打法是"大电池 + 长续航"。

所有头部企业都在拼谁的电池更大，谁的续航更长。市场的共识是：续航焦虑是电动车最大的痛点，解决续航就能赢得市场。

这个逻辑听起来很合理。
```

#### 2. 这家公司走的是哪条偏门（300-500字）
转折：但这家公司不这么做。

**要点：**
- 它在做什么不一样的事
- 这个选择在当时看起来如何（不被看好、被质疑）
- 它为什么这么选

**示例：**
```
这家公司选择了另一条路：不做大电池，做换电。

这个选择在 2018 年看起来很不明智。换电模式在行业里已经被证明"不可行"——Better Place 在 2013 年破产，特斯拉也放弃了换电。

但这家公司的创始人有不同的判断。他认为，换电模式之所以失败，不是因为模式本身有问题，而是因为时机不对、场景不对。

他选择了一个更窄的场景：出租车和网约车。
```

#### 3. 偏门为什么反而可能是机会（400-600字）
这是文章的核心：解释为什么"偏门"可能是对的。

**要回答的问题：**
- 主流路线有什么问题/局限？
- 偏门路线解决了什么主流路线解决不了的问题？
- 偏门路线的前提条件是什么？这些条件现在是否成立？

**写作要点：**
- 不要只说"它是对的"，要说"为什么它可能是对的"
- 用逻辑推演，不要只用结果证明
- 承认不确定性

#### 4. 谁低估了它（200-300字）
说明市场对这家公司的误解。

**常见误解：**
- 把它归类到错误的赛道
- 用主流路线的标准评价它
- 忽视了它的独特价值

#### 5. 谁会后来追上来（200-300字）
不要只说好话，要说竞争。

**要回答的问题：**
- 如果这条路被证明是对的，谁会跟进？
- 这家公司有什么先发优势？
- 先发优势能维持多久？

#### 6. 现在是不是好时点（200-300字）
结尾，给出判断。

## 模型 D：公司气质型

### 适用场景
- 公司有独特的管理风格或组织文化
- 这种气质深刻影响了公司的产品、战略、财务表现
- 适合写成"这家公司像谁"

### 文章结构

#### 1. 这家公司像谁（200-300字）
开场用一个类比，让读者快速建立认知。

**示例：**
```
如果要用一个类比来形容这家公司，我会说：它像钨行业的紫金矿业。

什么意思？

紫金矿业的特点是：在所有人都在找大矿的时候，它在收小矿；在所有人都在拼品位的时候，它在拼成本控制。

这家公司也是这样。在所有人都在做高端市场的时候，它在做中低端；在所有人都在拼技术参数的时候，它在拼交付能力。
```

#### 2. 为什么这么像（300-500字）
解释这个类比背后的逻辑。

**要点：**
- 管理层的背景/思维方式
- 公司的组织结构/决策机制
- 公司的资源禀赋/约束条件

#### 3. 它的管理风格决定了什么（400-600字）
说明管理风格如何影响公司的战略和执行。

**常见影响：**
- 集权 vs 分权 → 决策速度、创新能力
- 工程师文化 vs 销售文化 → 产品导向 vs 市场导向
- 激进 vs 稳健 → 扩张速度、财务杠杆

#### 4. 它的组织方式如何影响财务和产品（300-500字）
从组织到结果的传导路径。

**示例：**
```
这家公司的组织方式很特别：它不设事业部，所有产品线都在一个大部门下。

这种组织方式的好处是：资源可以快速调配。当某个产品线需要支持时，其他产品线可以立刻抽调人力。

但坏处是：产品线之间的边界不清晰，容易出现内部竞争和资源争夺。

这种组织方式反映在财务上，就是：毛利率波动大，但周转率高。
```

#### 5. 它的强项和短板是什么（200-300字）
基于气质，说明公司的优势和劣势。

#### 6. 它适合哪类投资者/读者关注（200-300字）
结尾，给出判断。

## 模型 E：争议拆解型

### 适用场景
- 市场对公司存在明显分歧
- 多头和空头都有各自的逻辑
- 适合写成"市场为什么吵起来"

### 文章结构

#### 1. 市场为什么吵起来（200-300字）
开场说明争议的背景。

**示例：**
```
最近这家公司的股价很有意思：一边是机构在加仓，一边是散户在抛售。

机构说它是"被低估的成长股"，散户说它是"高估值的周期股"。

谁对谁错？
```

#### 2. 多头在讲什么（300-500字）
客观呈现多头的逻辑。

**要点：**
- 多头的核心论点
- 支撑论点的数据/事实
- 多头的假设前提

#### 3. 空头在讲什么（300-500字）
客观呈现空头的逻辑。

**要点：**
- 空头的核心论点
- 支撑论点的数据/事实
- 空头的假设前提

#### 4. 真正该看的不是哪一边声音大（300-500字）
这是文章的"洞察"部分：争议的本质是什么。

**常见本质：**
- 对同一个数据的不同解读
- 对未来的不同假设
- 对风险的不同权重

#### 5. 真问题在哪（200-300字）
指出争议背后真正需要回答的问题。

**示例：**
```
多头和空头争论的焦点是：这家公司的高增长是周期性的还是结构性的。

但真正的问题不是这个。真正的问题是：即使是结构性增长，这个增长能维持多久？

因为这个行业的特点是：技术迭代快，先发优势容易被后来者追上。
```

#### 6. 我的判断（200-300字）
结尾，给出明确判断。

**要点：**
- 不要和稀泥，要有态度
- 说明判断的前提和假设
- 说明如果判断错了，最可能错在哪里

## 模型选择决策树

```
公司特点分析
  ↓
创始人/董事长有强烈个人特质？
  → 是 → 模型 A：人物下注型
  ↓ 否
公司经历了重大转折/重组/借壳？
  → 是 → 模型 B：公司命运型
  ↓ 否
公司走的是行业非主流路线？
  → 是 → 模型 C：行业裂缝型
  ↓ 否
公司有独特的管理风格/组织文化？
  → 是 → 模型 D：公司气质型
  ↓ 否
市场对公司存在明显分歧？
  → 是 → 模型 E：争议拆解型
  ↓ 否
回到模型 A 或 D（通用性最强）
```

---

## 混合使用

实际写作中，可以混合使用多个模型。例如：

- **主线用模型 A（人物下注型）+ 穿插模型 E（争议拆解）**
  - 适合：创始人有强烈个人特质，且市场对公司有争议
  - 结构：用人物下注型讲故事，在关键节点穿插市场争议

- **主线用模型 B（公司命运型）+ 穿插模型 C（行业裂缝型）**
  - 适合：公司经历转折，且转折后走的是非主流路线
  - 结构：用公司命运型讲转折，用行业裂缝型讲新路线

**原则：主线只用一个模型，穿插不超过一个模型，否则结构会乱。**
