# 公众号写作 7 个技巧（详细版）

> 本文件是 SKILL.md 第 4 节「撰写正文」的详细展开。
> SKILL.md 保留速查摘要，需要具体示例时读本文件。
>
> **核心原则**：每 300-500 字必须有一个"钩子"，让读者继续往下读。

---

## 技巧 1：开头 3 段必须有反差或悬念

不要平铺直叙，要制造"咦，有意思"的感觉。

✅ 好的开头：
```
2025年12月，中钨高新花了一大笔钱收购了衡阳远景钨业。
市场当时不太理解：钨价还在周期底部徘徊，为什么要在这个时候买矿？
三个月后，答案出来了。2026年一季度，中钨高新净利润预增超250%。
```

❌ 差的开头：
```
中钨高新是一家专注于钨产业链的上市公司，主营业务包括钨矿开采、冶炼和硬质合金制造。
公司成立于1996年，隶属于中国五矿集团。
```

---

## 技巧 2：每一大段都要推动一个问题

不要只陈述事实，要让读者产生"然后呢？"的好奇心。

结构模板：
1. 抛出一个问题或反常识现象
2. 给出答案或解释
3. 引出下一个问题

示例：
```
为什么别人不这么做？（问题）
因为全产业链太重了。矿山投资大、周期长、管理复杂。（答案）
但李仲泽赌的就是这个"别人不做"的事。（引出下一个问题：他为什么敢赌？）
```

---

## 技巧 3：用"时间锚点"制造节奏感

长文容易让人失去方向感。用时间节点作为章节分隔，让读者知道"故事进展到哪了"。

示例结构：
- 2016年：确立战略
- 2024年：第一次收购
- 2025年：第二次收购
- 2026年：业绩爆发

每个时间节点都是一个小高潮，读者会想知道"下一个节点发生了什么"。

---

## 技巧 4：数据要"说人话"，不要堆表格

❌ 差的写法：
```
| 年份 | 营收 | 净利润 | ROE |
|------|------|--------|-----|
| 2023 | 127.4亿 | 5.87亿 | 8.69% |
| 2024 | 147.4亿 | 10.33亿 | 14.11% |
| 2025 | 176.4亿 | 12.81亿 | 13.32% |
```

✅ 好的写法：
```
2023年，钨价低迷，中钨高新净利润只有5.87亿。
但它没有停下来，反而在这个时候收购矿山。
2025年，钨价上涨，净利润达到12.81亿，同比增长29%。扣非净利润更是暴增585%。
```

关键数据融入叙事，让数据成为故事的一部分。

## 技巧 5：每 500 字插入一个"小高潮"

小高潮可以是：
- 一个反转（"市场这才反应过来"）
- 一个判断（"这就是全产业链的价值"）
- 一个类比（"像钨行业的紫金矿业"）
- 一个反问（"为什么这条路走通了？"）

这些小高潮是"呼吸点"，让读者不会觉得文章太长。

---

## 技巧 6：用"三段式"控制每个章节长度

每个大章节不超过 800 字，内部分为 3 小段：
1. 抛出问题（100-150字）
2. 展开分析（400-500字）
3. 给出结论（100-150字）

示例：
```
## 为什么这条路走通了？（问题）

全产业链不是新概念，但能做成的不多。中钨高新为什么能？

**第一，成本优势。**（展开1）
当你同时拥有矿山、冶炼、合金三个环节，你的成本结构就和别人不一样了...

**第二，抗周期能力。**（展开2）
钨是周期性商品，价格波动很大...

**第三，高端制造溢价。**（展开3）
很多人以为中钨高新是资源股，但如果你看它的业务结构...

这就是全产业链的价值。（结论）
```

---

## 技巧 7：结尾要有"未完待续"感

不要写成"总结陈词"，而是留一个悬念或判断，让读者觉得"这个故事还没结束"。

✅ 好的结尾：
```
真正的考验是：当钨价再次进入下行周期时，中钨高新能不能依然保持增长。
如果能，那它就真的成了钨产业链的"链主"。
```

❌ 差的结尾：
```
综上所述，中钨高新是一家值得关注的优质公司，投资者可以持续跟踪。
```

---

## 行文细节规则

- 每段 3-5 句话，不超过 150 字
- 多用句号，少用逗号串联
- 少用术语堆叠，必要时用类比解释
- 避免"综上所述""总而言之"等总结词
- 每一大段都要推动一个问题，不做平铺直叙
