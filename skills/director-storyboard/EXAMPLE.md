# 使用示例

## 基础版使用（日常生成）

```
请加载技能：director-storyboard-core.md

以下是我的剧本：

[在这里粘贴你的剧本内容]
```

## 进阶版使用（精修阶段）

```
请追加技能：director-storyboard-advanced.md

请基于以上进阶规则，对已生成的分镜脚本进行精修。
```

## 全量加载（一次性生成高质量版本）

```
请同时加载以下技能：
1. director-storyboard-core.md
2. director-storyboard-advanced.md

以下是我的剧本：

[在这里粘贴你的剧本内容]
```

## 示例剧本

```
【场景：废弃工厂，黄昏】

李明推开锈迹斑斑的铁门，警惕地环顾四周。远处传来金属碰撞的声音。

他握紧手中的手电筒，缓缓向声音来源走去。

突然，一个黑影从天而降，李明本能地闪避，手电筒掉落在地。

黑影站起身来，月光下露出一张熟悉的脸——是他失踪多年的搭档，张伟。

"你还活着？"李明难以置信。

张伟沉默地看着他，眼神复杂。

"对不起，我不得不这么做。"张伟说完，转身消失在黑暗中。

李明捡起手电筒，望着张伟离去的方向，陷入沉思。
```

## 注意事项

1. **镜头数量**：500字剧本通常会生成60-120个镜头
2. **题材影响**：动作/战斗场景镜头更密集，对话/氛围场景镜头较少
3. **衔接逻辑**：场景跳转处必须使用复合标签（如：情绪因果+视线引导）
4. **质量检查**：生成后请对照自检清单逐项检查
