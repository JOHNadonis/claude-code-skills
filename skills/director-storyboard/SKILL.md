# AI导演分镜脚本 Skill

将任意剧本转化为工业级逐镜头导演分镜脚本。

## 触发条件

当用户提到以下关键词时激活：
- 分镜脚本
- 导演分镜
- 剧本转分镜
- storyboard
- 镜头脚本
- 分镜设计

## 功能

- 将剧本转化为可直接投入工业化生产流程的、逐镜头的导演级分镜脚本
- 1-2分钟成片 → 通常产出60-120个镜头
- 500字剧本 ≈ 1分半时长

## 使用方法

### 基础版（日常生成）
```
加载技能：director-storyboard-core.md
[粘贴剧本]
```

### 进阶版（精修阶段）
```
追加技能：director-storyboard-advanced.md
请基于以上进阶规则，对已生成的分镜脚本进行精修。
```

### 全量加载
```
加载技能：director-storyboard-core.md + director-storyboard-advanced.md
[粘贴剧本]
```

## 输出格式

表格输出，包含8列：
- 镜头编号
- 关联剧本
- 景别/视角/构图
- 画面描述
- 情绪&节奏
- 声音设计
- 衔接逻辑
- 运镜提示

## 核心流程

1. **剧本解读与节拍拆解** - 以情绪节拍为最小单位
2. **前置建构** - 5-15个无台词视觉引入镜头
3. **角色建立** - 每个主要角色的人设镜头
4. **逐镜头创作** - 多景别、多视角、多元构图
5. **衔接逻辑设计** - 标注衔接逻辑标签
6. **节奏控制** - 张弛有度，引导观众情绪流
7. **质量自检** - 7项核心检查

## 衔接逻辑标签

动作顺接 | 情绪因果 | 视线引导 | 台词驱动 | 音效桥接 | 同框过渡 | 焦点转换 | 角色调度 | 匹配转场 | J-Cut | L-Cut | 逻辑因果 | 对比反差 | 节奏硬切

## 注意事项

- 镜头数量是细化原则自然落地的结果，不为凑数而增镜
- 动作/战斗密集型剧本偏向上限，对话/氛围型剧本偏向下限
- 场景/情绪跳转处必须标注复合标签
- 关掉声音能否理解至少60%的故事和情感
