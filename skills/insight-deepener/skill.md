## 理论框架：七表本体论

基于维特根斯坦逻辑哲学与奥地利学派经济学，用7张表系统分析商业现象：

### 1. 对象表（Objects Table）

识别构成现象的最基本、不可再分的实体

- 人的对象：行动人、企业家、消费者
- 偏好对象：主观价值、目标
- 时间对象：时间偏好、跨期选择
- 知识对象：分散知识、信息

### 2. 事态表（States of Affairs Table）

将复杂现象分解为最基本的原子事态（对象的组合）

- 评价事态：人 + 物品 + 偏好 → 价值判断
- 选择事态：人 + 选项 + 偏好 → 机会成本
- 发现事态：人 + 知识 + 时间 → 企业家发现
- 生产事态：人 + 资源 + 时间 → 转化

### 3. 复合事态表（Complex States of Affairs Table）

展示原子事态如何通过逻辑运算符组合成复杂过程

- 序列型（→）：事态A → 事态B → 事态C
- 并行型（∧）：事态A ∧ 事态B ∧ 事态C
- 条件型（⇒）：IF 事态A THEN 事态B

### 4. 关系表（Relations Table）

揭示对象与事态之间的结构性联系

- 因果关系、对立关系、转化关系、依赖关系、替代关系

### 5. 规则表（Rules Table）- 重点产出金句

从行动公理演绎出支配该现象的经济规律

- 价值规律：边际效用递减律、主观价值论
- 时间规律：时间偏好律、利息存在律
- 市场规律：利润信号律、竞争过程律
- 知识规律：知识分散律、价格信号律

### 6. 形式表（Forms Table）

识别反复出现的结构模式

- 投资形式：目的 → 发现 → 配置 → 风险 → 收益
- 消费形式：偏好 → 选择 → 支付 → 满足
- 竞技形式：规则 → 训练 → 竞争 → 胜出

### 7. 定义表（Definitions Table）- 重点产出金句

为所有核心概念提供严格的、基于行动公理的定义

- 投资、消费、竞技、上学vs读书、伪投资等

## 工作流程

### 输入

用户提供一段观点或想法

### 处理

1. **快速七表分析**：用七表框架系统分析（不需要完整输出，只需要内部分析）
2. **提取规律**：从规则表中提取支配性规律
3. **严格定义**：从定义表中提取核心概念定义
4. **产出金句**：从规律和定义中提炼20-30条金句
5. **分类整理**：按7个类别整理金句

## 金句产出标准

### 1. 理论支撑

每条金句都必须有明确的理论支撑（规律或定义），不是空洞的口号

### 2. 可直接使用

金句可以直接用于文稿，不需要再加工

### 3. 多样性

提供不同风格的金句：

- 强调行动的
- 强调认知的
- 强调对比的
- 强调数据的
- 强调类比的

### 4. 分类清晰

按7个类别整理，方便用户选择：

- 类别1：核心概念定义
- 类别2：目的-手段链条
- 类别3：时间与价值
- 类别4：机会成本与替代
- 类别5：稀缺性与转移
- 类别6：对比与区分
- 类别7：陷阱与误区

## 注意事项

- **不要过度分析**：七表分析只是内部工具，不需要完整输出，重点是产出金句
- **保持简洁**：每条金句控制在1-2句话，不要冗长
- **理论支撑**：每条金句都要标注理论支撑，增强说服力
- **多样性**：提供不同风格的金句，让用户有选择空间
- **可用性**：金句可以直接用于文稿，不需要再加工
