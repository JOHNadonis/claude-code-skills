# JD 示例 — 弱电智能化解决方案架构师

> 用于测试 jd-resume-builder skill 的示例岗位描述

---

## 岗位职责

1. 负责智算中心机房基础设施的整体方案设计，确保为IT设备提供安全、可靠、高效的动力和环境支持。
2. 专业系统设计：电气系统（供配电、UPS、配电柜、柴油发电机）、制冷系统（精密空调、冷热通道隔离、液冷方案）、弱电智能化系统（综合布线、动环监控、安防、消防、DCIM）。
3. 进行机房规划、空间布局、承重核算、PUE值测算，确保方案符合国标A级/Tier III及以上标准。
4. 参与项目前期技术交流、编制技术方案、图纸深化及工程量清单，支持投标和项目实施。
5. 与电力、制冷、弱电等设备供应商及工程分包商进行技术对接和管理。

## 岗位要求

1. 统招本科及以上，电气工程、暖通、自动化、智能化、机电等相关专业。
2. 5年以上大型数据中心或IDC机房基础设施规划设计经验，具备A级数据中心项目全程经验者优先。
3. 精通数据中心国家标准（GB50174-2017）及国际标准（TIA-942、Uptime Tier）。
4. 深刻理解电气系统架构、UPS、HVDC、柴油发电机等关键产品；精通制冷技术（风冷、水冷、液冷）。
5. 熟练使用AutoCAD，具备文档编写和成本意识。
6. 加分项：注册电气工程师、机电工程类证书或Uptime ATD/ATS认证。
