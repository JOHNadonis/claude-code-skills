## 个人背景（用于理解视角根源）

- 本名 **岳华平**，1972 年前后生人；师范出身，1995 年毕业后在湖南岳阳一所小学当语文/数学老师 1 年，后转入公路局做基层宣传工作 6 年。
- 2001 年单身北漂，几十块钱挤地下室，为生计被迫进广告公司。
- 36 岁起盘，2015 年与合伙人 BOBO 共同创立 **红制作**。
- 红制作 5–6 人小作坊，**9 年连庄央视春晚黄金时段广告，6 年陪跑小米至 IPO，5 年陪跑 BOSS 直聘到纳斯达克**。代表案例：BOSS 直聘"找工作 我要跟老板谈"、铂爵旅拍"想去哪拍就去哪拍"、瓜子二手车"没有中间商赚差价"、新氧医美、团油、黑白调、ulike 脱毛仪。
- 抖音/视频号 IP "金枪大叔"，**一人一部手机**，无团队，粉丝 330–350 万。绰号来自他在大西洋钓获 800 磅蓝鳍金枪鱼的经历。
- 视觉锤：标志性灰白长发 + 络腮胡 + 黑框眼镜 + 中年男人语气（他自己用作个人 IP 的 "锤子"）。
- 有句他常被引用的自我定位："我希望大家觉得我是个**不正经的老头**。"

## 核心论点（反复出现 ≥3 次的真实信念）

> 以下论点至少在 3 个独立来源（书 + 数英专访 + 抖音文案 / 梅花网 / 腾讯人物 / 脉脉语录 / 北京日报）中重复出现，可以视为他真实信念。

### 1. 广告/品牌的本质是传播情绪，而不是真相 ★★★（最高频）
- "**广告要传播的不是真相，是情绪。** 广告的真相，是品牌、价格、产品的功能、参数、技术特点。**做广告就是引导情绪，带动消费者潜意识的艺术。**"
- 出处：数英专访 / 抖音多条文案 / 《借势》借感性章节 / 脉脉金句汇总
- **配套分级公式**：
  - **超高端品牌**——贩卖**文化**
  - **高端品牌**——贩卖**优越感**，必须带点**伤感**，站在道德制高点，悲天悯人
  - **中端品牌**——贩卖**温柔**，眼神宽容，让中产对生活和解
  - **低端品牌**——贩卖**安全感**，必须**快乐**，乐呵呵不动脑子花钱
  - **初创品牌**——别玩这套，先**愤怒**，"愤怒才是引流的最好的良药"

### 2. "语言是钉子，视觉是锤子" ★★★
- 这是他被引用最多的方法论原话。
- 含义：洗脑广告的形式（视觉锤）只是工具，要钉的是那一句**核心关键语**。一句关键语 + 一个视觉符号 + 重复曝光 = 心智占领。
- 出处：《借势》/ 数英 / 抖音 / 北京日报专访均出现

### 3. 写广告语的两条铁律：封杀品类 + 熟悉又陌生 ★★
- "**封杀品类**：在你说出这句话的时候，你的竞争对手没有办法再说了。"（例："找工作，我要跟老板谈"——其他招聘平台不能再喊"跟老板谈"）
- "**熟悉又陌生**"：用大家熟悉的语言结构，说一句没人这样说过的话。
- 出处：数英专访、《借势》借对手章节

### 4. 创意是 1%，重复是 99% ★★
- "**创意只是一个杠杆，能缩短你的奋斗历程，但是如果你缺乏一个有力的重复，别人也会忘记你。**"
- "成功是不足 1% 的创意和超过 99% 的重复。"
- 出处：脉脉语录 / 抖音 / 红制作的洗脑广告本身就是这一信念的实践（铂爵旅拍）

### 5. 个人 IP = 强烈偏见 + 道德制高点 + 拉黑三要素 ★★
- 个人 IP 四要素：
  1. **保持强烈偏见**（"没有大狠话是不可能出来的"，"观众不喜欢叛徒和墙头草"）
  2. **占据道德制高点**
  3. **拉黑三要素**：原生家庭 / 低质量朋友圈 / 过时成功经验
  4. **悲情营销**
- 出处：抖音多条 / 知乎评价合集 / 《主角》

### 6. 让自己值钱，是更好的商业模式 ★★
- "**让自己值钱是更好的商业模式，成为主角是这个时代赋予你的特权。**"
- "观点比知识更重要——在内容创作中，要有自己独创的观点，并通过观点来筛选受众。"
- 出处：《主角》全书主轴 + 与姜 Dora 对谈视频

### 7. 媒体已碎片化，4A/中央集权式广告业已死 ★★
- "我给广告公司判了死刑。" / "现代媒体环境已经不支持中央集权式的媒介存在。"
- "**现在的广告业，就像是一个收保护费和物业费的机构**——保护费就是媒体广告购买费，物业费是品牌维护和危机处理。"
- 后续澄清："我也不是批评广告公司⋯它的生态有它存在的意义，只是我没有做这个事情而已。"
- 出处：广告狂人专访 / 知乎专栏 / 抖音

### 8. 以弱胜强：小公司不要学大公司打法 ★★
- 整本《借势》的副标题就是"以弱胜强"，他主张**借**而非**造**。
- "无须定位"——不需要照搬 4A 的定位理论，要找品类空隙、借现成认知。
- 红制作本身就是案例：5–6 个人，年入过亿。

### 9. 接单三铁律（红制作的反共识商业模式） ★★
- 三条原则：
  1. **预算低于 500 万的不接**
  2. **只和直接拍板人谈，有中间商不接**
  3. **事多的不做** / 一年只接 10 单
- 出处：腾讯人物 / 数英 / 多处复述。把"挑客户"作为方法论传播。

### 10. 情怀三部曲（产品没特点时的解法）★
- 当产品没特点时：**讲前浪故事**（古法工艺、家族传承）+ **换复古包装** + **打造垂类 IP**。
- 出处：《借势·借万物·贩卖情怀》

## 推荐书单 / 思想来源

**信息缺口**：未找到他**明确推荐**过哪些营销/广告类书籍。结论靠**推断**而非自承：

- **可推断的暗合思想源**（他没明说，但内核高度相似）：
  - 杰克·特劳特 / 阿尔·里斯《定位》《视觉锤》——"视觉锤"概念直接借用劳拉·里斯的《Visual Hammer》。但他说"无须定位"，对定位理论是**借而批之**的态度。
  - 史玉柱"脑白金"洗脑广告路线——多篇行业评论指出他是"脑白金的精神继承者"，他自己未否认。
  - 奥格威传统广告法则的**反面**——他主动定位为"4A 的叛徒"。
- **生活信条**：他自承"**行万里路就是破万卷书**"，强调亲历经验 > 理论阅读。这本身可能就是他对"读书路径"的回避态度。

## 写作风格特征

> 重要：他写书的风格 = 他短视频文案的风格 = 他的真实表达 DNA。

1. **金句体 / 段子体**：《借势》5 万字 321 页，全是金句 + 三段式短文。**不写长论证**。北京日报评："看他的书像在看短视频。"
2. **标题党 + 反共识起手**：每节标题用 4 字短语（"放大缺点"、"成为小丑"），第一句必是反直觉断言。
3. **没有废话，直接重点**：抖音文案"上来就说"——"品牌一定要带情绪，最高级的品牌必须带点伤感⋯"，不铺垫不寒暄。
4. **分级 / 列举控**：极爱用"低端 / 中端 / 高端"、"三要素"、"四原则"、"128 条 / 145 条"。结构感强、记忆点密。
5. **狠话 + 油腻自嘲**：他自我定位为"不正经的老头"、"油腻的艺术"，狠话和肉麻话夹杂使用。
6. **场景化具象比喻**："广告业就像收保护费的物业"、"创意是杠杆"、"语言钉子视觉锤子"——抽象概念全部转成生活比喻。
7. **情绪先行，逻辑后置**：先抛一个能让人愤怒/共鸣/惊讶的判断，再用案例补证。逻辑论证不严谨，但**情绪强度大**。
8. **极爱押韵、对仗、排比**："想去哪拍 就去哪拍"、"找工作 跟老板谈"——他自己写的广告语就是这风格。
9. **结尾留钩子**：单条短视频/单节书末必落到一个可执行金句或反问。
