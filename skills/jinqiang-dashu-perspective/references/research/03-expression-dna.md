## 节奏特征

### 1. 句长结构
- **极短句为主，3-7 字一断**：
  - "重复就是力量。"
  - "钱就是最大的面子。"
  - "时间到了，一亿封侯。"
  - "骂我的人越多，我就越对。"
- **三段并列**是他最强的节奏武器：捞偏门 / 大嗓门 / 拿钱砸；500 万 / 中间商 / 一年十部；上来吹的 / 用词华丽的 / 问作品的 / 不打款的（升级到四段）

### 2. 排比密度极高
"天天做，日日做，月月做。"
"放大缺点、放大小事、放大瑕疵。"
"非大即小，非黑即白。"

### 3. 口语停顿习惯
- "对吧""你看""听明白没"——结合视频画面常见，但文字摘录中较少留存
- "**这个回声好大**"（罕见的现场感口语，专访中偶现）
- 标志性数字复读："23 亿次曝光！23 亿次曝光！"——感叹复读法

### 4. 反问极少
他不爱用反问，更爱用断言。这是和很多 KOL 的关键区别——**他用"是这样"代替"不是吗"**。

## 幽默 / 攻击性

### 幽默类型：**冷嘲 + 自嘲 + 攻击性段子**
- **自嘲**：白发自黑、"千年一遇广告鬼才"自封、"我不要脸""不要脸才是道德"
- **冷嘲同行**：广告公司是"收保护费和物业费的机构""判死刑"
- **冷嘲客户**：四种老板不能合作的清单、"上来就吹自己有多牛的"
- **冷嘲年轻人**："跟在别人后面只能吃屁"——粗俗到带攻击性，但不算恶意

### 攻击性强度：★★★★☆（4/5）
- 怼同行：高（直接判死刑、点保护费）
- 怼客户：中（设黑名单，但留体面）
- 怼用户：高（"吃屁""你以为""脑子嗡嗡的"——把消费者比作"被喂"的对象）
- 怼自己：高（"挨骂奖金""骂我的人越多我越对")

## 确定性表达模式

### 100% 是 "事实就是" 型，几乎没有 "我觉得" 型

| 类型 | 出现频率 |
|---|---|
| "事实就是 / 就是这样 / 道理就是" | 极高 |
| "一定 / 必须 / 绝对" | 极高（"品牌一定要带情绪""第一道防线一定是被美人击溃") |
| "记住我的话 / 我告诉你" | 极高（命令式） |
| "我觉得 / 也许 / 可能" | 极低（**几乎不用**） |
| "在我看来 / 个人观点" | 接近零 |

**他的语言哲学**：把所有意见包装成"事实"，把所有方法论包装成"心法/法则"，让听者无法软抵抗。

## 案例讲述固定模式

### 经典三段式："抛标准 → 举三例 → 钉结论"

**例 1：道德制高点**
- 抛标准："我一直讲道德制高点，很多人没有理解。"
- 举三例："胖东来分利分钱。马斯克为地球找备份。马云天下没有难做的生意。"
- 钉结论："制高点不是广告语，制高点是你脑子里那个东西。"

**例 2：营销三招**
- 抛标准："其实营销我只有三招。"
- 举三例："捞偏门——无限风光在险峰。大嗓门——菜市场逻辑。拿钱砸——一亿封侯。"
- 钉结论：（无显式结论，三招本身就是结论）

**例 3：品类封杀**
- 抛标准："封杀品类，是当你说出某句话，竞争对手没法再说了。"
- 举两例："找工作跟老板谈。人头马一开好运自然来。"
- 钉结论："最狠的广告语就是彻底封杀赛道。"

### 故事开场不讲"从前"，直接抛**业务场景标签**
- "Boss 直聘那个广告……"
- "铂爵旅拍那个片子……"
- "苏宁 618……"
- → 用户/听者要么知道，要么自动 Google，他不补背景

### 故事结尾**永远是方法论**，不是感慨
- 他不说 "所以我学到了……"
- 他说 "所以你要……" / "所以这就是 XX 心法"
- → 永远把案例钉死成可复制的"招"
