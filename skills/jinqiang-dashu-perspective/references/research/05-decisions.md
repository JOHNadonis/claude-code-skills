# 金枪大叔 - 决策与行动研究

> 资料源：digitaling、广告门、梅花网、36氪、腾讯新闻、澎湃新闻、知乎专栏、博集天卷、网易、北京日报等第三方报道。
> 时间窗口：2001 北漂 → 2025《主角》新书。
> 取材原则：行动 > 言论；第三方记录 > 自述；可验证数据优先。

## 3. 案例的共同模式

1. **押宝"被骂红"路线**：Boss 直聘、铂爵、Ulike、五个女博士共用同一套引擎——重复+情绪+争议=记忆点。他公开说"广告不传播真相，传播情绪"。
2. **只接 0→1 阶段品牌**：客户多是想冒头的中小品牌，预算饿狼型；品牌方愿赌服输。这降低了"长期品牌资产受损"对他的反噬——品牌赢了感谢他，输了他换下一个。
3. **借大势**：选热点档期（春晚、世界杯、双 11）+ 选热点客户（要 IPO、要融资）。他的方法论书名直接叫《借势》，行动一致。
4. **只对接老板，不对接 marketing**：跳过审美最严的中间层。这是他能持续推出"被骂红"作品的组织前提。
5. **不打透产品本身，打透一句话**：他的所有出圈案例都靠一句 slogan + 视觉锤，而非长内容。

## 4. 言行一致性检验

**一致的部分（言行同步）：**
- 说"做高客单"→ 红制作单价 500 万起，咨询费 50 万起，**真做了**。
- 说"不带货、不卖课、不直播"→ 抖音 400 万粉至 2025 年仍未直播带货，**真做了**。
- 说"广告要带情绪"→ 所有出圈作品确实是情绪驱动，**真做了**。
- 说"放大缺点"→ 自己的人设（白发、乌尼莫克、不上大学）就是把"非主流标签"放大成视觉锤，**真做了**。

**矛盾点（说一套、做一套）：**
- 说"做有尊严的广告"，却出品物化女性、贩卖焦虑的作品（Ulike、五个女博士）。**行动证明：尊严让位于效果**。
- 说"不卖课、不收割粉丝"，但出书、收 50 万咨询费、签新书巡回——**只是绕开了"课程"标签的知识变现**。
- 说"看不上同行，给广告公司判死刑"，但事后改口"我们和同行之间是和解的"。**有争议时进，有压力时退**。
- 嘲讽"传统广告公司是物业公司收保护费"，但红制作本身就是吃甲方预算的传统广告公司，只是甲方更头部。

## 5. 关键转折点

| 时机 | 行业变化 | 他的反应 | 反应快慢 |
|---|---|---|---|
| 2013 移动互联网 | 互联网公司广告预算外溢 | 创立红制作切入互联网客户 | 准时上车 |
| 2018 抖音崛起 | 短视频平台爆发 | 2020 才开账号 | **慢 1.5–2 年**，但靠"广告大佬"身份后发拉粉 |
| 2020 直播带货 | 罗永浩、董宇辉模型 | **明确拒绝**，坚持纯内容 | 主动不上车，反而成差异化护城河 |
| 2021 私域流量 | 微信生态变现 | 不参与，未做社群、未做付费圈 | 主动不上车 |
| 2022 视频号崛起 | 平台红利窗口 | 同步铺，但不重押 | 跟跑 |
| 2023 监管收紧（女博士事件） | 广告法/反低俗 | 红制作未公开道歉，让品牌方独自背锅 | 风险转嫁 |
| 2024–2025 知识付费降温 | 头部博主收割见顶 | 出书《主角》替代卖课 | 用「出版」绕过「卖课」骂名 |

**抓住的：** 移动互联网 0→1 客户、电梯+春晚渠道红利、抖音个人 IP、出版变现。
**错过的（或主动放弃的）：** 直播带货、私域、付费社群、海外业务、AI 营销工具化产品。
**信号：** 他的"错过"几乎都是「重运营、重交付、重客服」的赛道。他只做「一次性创意收费」的高毛利生意。

## 6. 错赌与对赌

**押对的：**
1. 押"洗脑广告在中国还有 10 年红利"——对了。
2. 押"个人 IP > 公司品牌"——对了，红制作没账号，他一个人 400 万粉。
3. 押"不接小单"——逆势策略，在低端被同行卷死时他已经在头部躺赢。
4. 押"不下场带货"——保住了"广告大佬"权威定位，没有被算法和退货率拖垮。

**押错的（或代价的）：**
1. 风格红利依赖严重——五个女博士事件是这套打法的天花板，再用就要踩监管线。
2. 个人 IP 没有可继承性——红制作公司价值约等于他个人；一旦他不做账号，公司估值会折损。
3. 没有第二曲线——课程、SaaS、AI、海外都没押。年入过亿全靠"创意+IP"两条腿，一旦审美换代会接不住。
4. 履历叙事过度依赖客户成功——"陪跑小米/Boss 直聘到 IPO"长期不可持续，因为新一代客户少有这种 IPO 级标的。

## 7. 复盘风格

**承认错误？外归因？还是从不复盘？**

**结论：选择性复盘，强烈外归因，公开场合从不真正认错。**

证据：
- 五个女博士事件后，红制作未公开承认创意有问题，而是品牌方道歉、品牌方被罚。**红制作至今未对该案做公开复盘**。
- 谈到广告公司时先开炮"判死刑"，被反击后改口"和解"。**这不是复盘，是公关止损**。
- 公开内容里几乎全是"我当年怎么对的"——典型的幸存者偏差叙事。知乎评论指出"他讲的都是进攻的方法，问题是有幸存者偏差"。
- 早期"3 天被炒鱿鱼""第一份工作失败"反而被讲成励志故事——把失败包装为成长，不当成错误。

**他不复盘的内容：**
- 哪些客户合作翻车（不会主动讲）
- 哪些创意被毙了（不会主动讲）
- 哪些广告投放后无效果（不会主动讲）

## 8. 行动暴露的真实价值观

| 他常说的 | 行动暴露的真实排序 |
|---|---|
| "做有尊严的广告" | 效果 > 尊严。能被骂红就接 |
| "不为钱做广告" | 单价 500 万起，咨询 50 万起。**钱排前三** |
| "服务好客户" | 服务的是**老板个人**，不是公司组织 |
| "广告人要有理想" | 理想是「广告片→书→IP→商业地位」的升级路径 |
| "不收割粉丝" | 不卖课，但卖书、卖咨询、卖见面权 |
| "我没读过大学" | "中专"是放大缺点的人设武器，不是自卑标签 |
| "不抢同行饭碗" | 同行被他骂"物业公司"那一刻，他要的就是抢 |

**真实价值观排序（行动反推）：**
1. **稀缺性**（拒接、定高价、只见拍板人）
2. **个人 IP 资产**（控制叙事权、控制履历包装）
3. **效率**（5–6 人小团队、不做大公司）
4. **现金流安全**（不做需要持续运营/履约的产品）
5. **行业地位**（要被记住，哪怕被骂）
6. **客户结果**（重要，但不是第一）
7. **社会评价**（最低优先级，可被牺牲）

---

**核心判断：**

金枪大叔的真实形象是一个**精明的「稀缺性套利者」**——
- 用"中专+不卖课+不带货"做差异化定位
- 用"敢被骂"换取出圈效率
- 用"个人 IP + 小团队"做现金流闭环
- 用"复盘失败=讲励志"控制叙事权

他不是广告大师，是**把营销方法论用到自己身上、做得最彻底的人**。
他的"言行不一"恰恰证明他是顶级营销人——他知道**说什么**有传播力，也知道**做什么**有钱赚，并刻意保持二者错位。
