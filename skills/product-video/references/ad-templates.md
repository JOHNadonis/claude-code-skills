# AD 电商广告提示词模板

> 每个模板中的 `[方括号]` 内容需根据实际产品替换。
> 所有模板默认以 `@图片1` 引用用户上传的产品图。

---

## 模板 1：360旋转展示 (`360`)

**适用**：立体实物（杯子/瓶子/电子产品/化妆品瓶/3C产品）

```
@图片1中的[产品名]，360度高速旋转2圈后，突然停住蓄力分裂成了3个部分进行展示。随后分解后的[产品名]的上中下三部分快速向内旋转合成，一个完整的[产品名]。3D渲染产品展示特效，动感产品特效展示。
背景：[纯色/渐变/场景]干净背景，[暖光/冷光/自然光]照明。
禁止出现文字、logo、水印。
```

**简化版**（5秒以内用这个）：
```
@图片1中的[产品名]，在[背景描述]中缓慢360度旋转一圈，3D渲染产品展示，[光线描述]打光，质感细腻。
禁止出现文字、logo、水印。
```

## 模板 2：产品分解展示 (`explode`)

**适用**：多层结构产品（汉堡/三明治/分层化妆品/可拆卸产品）

```
将@图片1进行一个拆解，镜头保持静止，[产品名]悬浮在空中，开始旋转，[组件描述]轻柔而精准地分离，保持形状和比例，动作流畅，无任何额外效果，[逐层描述每个部件]都缓缓下落并完美地组合成一个完整的[产品名]。在此过程中，[质感细节描述]，保持着极致诱人的美感。
背景：[背景描述]。
禁止出现文字、logo、水印。
```

---

## 模板 3：角色带货 (`actor`)

**适用**：需要人物演示的产品（服装/配饰/美妆/食品试吃）

> 注意：不能上传真人正脸。角色需为虚拟形象/动漫/古装等非真实人物。

```
请根据@图片1的产品生成一段带货展示视频：
0-2s 一位[角色描述]的人物手持[产品名]，面带微笑展示产品正面。
2-4s 镜头推近，特写[产品名]的[关键卖点/细节]，[角色]用手指轻轻触碰展示质感。
4-6s 镜头拉远至中景，[角色][使用产品的动作]，表情从好奇转为满意。
6-8s [角色]手持[产品名]正对镜头，竖起大拇指。
音效：轻快的背景音乐，产品展示时的[相关音效]。
禁止出现文字、logo、水印。
```

## 模板 4：第一人称手作体验 (`firstperson`)

**适用**：食品/饮品/手工制作类产品

```
[时长]秒第一人称视角[产品类型]宣传广告，[品牌名]「[产品名]」，清新通透质感，柔调自然光，适配抖音/视频号信息流，画面聚焦手部动作与[产品]细节；
0-2秒：第一人称特写，你的手[制作动作1]，[音效描述]。
2-4秒：快速切镜卡点，你的手[制作动作2]，[视觉效果描述]。
4-6秒：第一人称成品特写，镜头拉近怼拍[成品细节]。
产品以@图片1为参考，保持产品外观一致性。
禁止出现文字、logo、水印。
```

## 模板 5：仿拍参考广告 (`clone`)

**适用**：有参考视频时，保持风格替换产品

```
参考@视频1的剪辑风格、运镜转场效果，
将视频中的原产品替换成@图片1的[新产品名]，
并调整视频内容使其重点突出[新产品名]的[核心卖点]，
保持@图片1产品一致性，以@图片1做为主体。
禁止出现文字、logo、水印。
```

## 模板 6：产品运镜卡点 (`montage`)

**适用**：通用产品展示，动感运镜配合节奏

```
以@图片1的[产品名]为主体，生成一段产品展示视频：
0-2s：产品从画面外滑入，低角度仰拍，[光线描述]打在产品表面。
2-4s：镜头环绕产品半圈，展示产品[侧面/背面]细节。
4-5s：镜头快速推近至产品[logo/标签/关键细节]特写，定格0.5秒。
整体节奏：前慢后快，最后一秒急停定格。
背景：[背景描述]，浅景深虚化。
禁止出现文字、logo、水印。
```

## 提示词组装检查清单

写完提示词后逐项确认：

- [ ] `@图片1` 引用了产品图？
- [ ] 有时间段划分？（至少 2-3 段）
- [ ] 镜头语言具体？（推近/拉远/环绕/特写，不是泛泛的"展示"）
- [ ] 产品名称和卖点明确？
- [ ] 有光线/背景描述？
- [ ] 时长与内容量匹配？（5秒别塞 5 个镜头）
- [ ] 末尾有禁止项？（文字/logo/水印）
- [ ] 全中文？
