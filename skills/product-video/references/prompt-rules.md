# Seedance 2.0 提示词规则速查

> 从官方教程 + 实测案例中提炼，写电商产品视频提示词时必须遵循。

## 7 条核心规则

### 1. 时间轴分段，逐秒编排

把视频拆成 2-3 个时间段，每段写清「画面 + 镜头」。

```
差: 一个保温杯在桌上旋转
好:
0-2s：低角度仰拍，保温杯从画面左侧滑入，金属光泽在灯光下闪烁
2-4s：镜头缓慢环绕半圈，展示杯身侧面的品牌标志
4-5s：镜头快速推近至杯盖特写，定格0.5秒
```

### 2. @引用明确

| 用途 | 写法 |
|------|------|
| 产品外观 | `@图片1中的[产品名]` |
| 首帧画面 | `@图片1作为首帧画面` |
| 仿拍运镜 | `完全参考@视频1的所有运镜效果` |
| 替换产品 | `将@视频1中的产品替换成@图片1` |

### 3. 专业镜头语言

用「镜头做什么」描述，不用「画面看到什么」：

| 类别 | 关键词 |
|------|--------|
| 景别 | 特写、中景、全景、超近景 |
| 运镜 | 推近、拉远、环绕、跟拍、摇镜 |
| 角度 | 俯拍、仰拍、低机位、鱼眼 |
| 节奏 | 快速硬切、慢推、急停、定格 |

### 4. 正面指令为主，负面指令兜底

先写「要什么」，再补「不要什么」：
- 正面: `动作流畅，产品质感细腻，光线柔和`
- 负面: `禁止出现文字、logo、水印`

### 5. 声音设计

即使是产品视频也要描述声音：
- `背景音：轻快科技感电子乐`
- `音效：金属碰撞声、液体倒入声`

### 6. 一致性约束

```
保持@图片1产品一致性
产品外观、颜色、品牌标识与原图完全一致
```

### 7. 全中文

Seedance 2.0 中文提示词效果远优于英文，所有提示词必须用中文。

## 禁止项模板

每个提示词末尾加上：
```
禁止出现文字、logo、水印。画面连贯流畅，无闪烁、无错位。
```

## 电商视频专用关键词

```
产品展示类：3D渲染产品展示、动感产品特效、质感细腻、光影流转
食品类：极致诱人、食欲感、新鲜感、汁水飞溅、热气腾腾
美妆类：丝滑质感、光泽感、柔焦效果、肤感展示
科技类：科技感、未来感、全息效果、粒子特效
通用：浅景深虚化、柔调自然光、高级感、电影级构图
```
