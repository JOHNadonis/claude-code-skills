# 分析框架 — 六维拆解模型

## 维度一：定位与人设

| 要素 | 分析问题 |
|------|---------|
| 垂直领域 | 做的是哪个细分赛道？宽还是窄？ |
| 人设标签 | 专家型/素人型/娱乐型/工具型？ |
| 差异化 | 同赛道里，这个账号的独特性是什么？ |
| 受众画像 | 谁在看？他们的核心痛点/欲望是什么？ |

**判断标准**：好的定位 = 受众能用一句话描述「这个账号是给XX人看的XX内容」

---

## 维度二：选题逻辑

### 选题矩阵分析

把账号内容按两个轴分类：

```
         高互动
            ↑
  流量型    |    爆款型
（引流）    |  （破圈）
←——————————+——————————→
  垂直型    |    转化型
（留存）    |  （变现）
            ↓
         低互动
```

- **爆款型**：互动量最高，通常是情绪共鸣或信息差内容
- **流量型**：搜索量大，靠关键词引流
- **垂直型**：专业深度，建立信任
- **转化型**：带货/引流私域，互动不高但有商业价值

### 高频选题词提取

从标题/封面中提取：
- 数字词（「3个方法」「7天」「100万」）
- 情绪词（「后悔」「震惊」「必看」「踩坑」）
- 身份词（「打工人」「宝妈」「00后」）
- 场景词（「下班后」「月薪5000」「35岁」）

## 维度三：内容结构

### 开头钩子类型（前3秒/前两行）

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| 痛点直击 | 「你是不是也有这个问题…」 | 情感/生活类 |
| 信息差 | 「99%的人不知道…」 | 知识/干货类 |
| 结果前置 | 「我用这个方法3个月涨了10万粉」 | 经验分享类 |
| 悬念制造 | 「最后一步我差点放弃了…」 | 故事类 |
| 身份共鸣 | 「作为一个XX，我…」 | 人设类 |

### 内容主体结构

常见结构：
- **问题-分析-解决**（干货类）
- **故事-转折-结论**（情感类）
- **列表式**（工具/资源类）
- **对比式**（测评/避坑类）

### 结尾 CTA 分析

- 引导评论：「你遇到过这种情况吗？」
- 引导收藏：「建议收藏，下次用得到」
- 引导关注：「关注我，持续更新」
- 引导私信：「想要资料包私信我」

## 维度四：视觉与封面

### 小红书封面规律
- 文字量：少（≤10字）vs 多（信息密集）
- 色调：暖色系/冷色系/高饱和/低饱和
- 人物：有人脸 vs 无人脸（有人脸通常 CTR 更高）
- 风格：生活感/专业感/极简/丰富

### 抖音封面/前3秒规律
- 是否有字幕钩子
- 场景：室内/户外/绿幕
- 节奏：快切/慢节奏

## 维度五：增长飞轮识别

### 常见飞轮模型

**内容飞轮**：
```
爆款内容 → 涨粉 → 更多曝光 → 更多爆款机会
```

**互动飞轮**：
```
高互动 → 平台推流 → 更多曝光 → 更高互动
```

**矩阵飞轮**：
```
主账号 → 引流子账号 → 私域沉淀 → 变现 → 反哺内容
```

### 关键转折点识别

寻找以下信号：
- 某条内容互动量是平均值的 5x 以上
- 粉丝量在短期内快速增长
- 内容风格/选题发生明显转变
- 开始出现广告/带货内容

---

## 维度六：变现路径

### 变现方式分类

| 方式 | 特征 | 适合阶段 |
|------|------|---------|
| 平台广告分成 | 被动收入，需要高播放量 | 成熟期 |
| 品牌合作 | 主动收入，需要垂直影响力 | 成长期+ |
| 带货佣金 | 需要信任度+选品能力 | 成长期+ |
| 知识付费 | 需要专业背书 | 任何阶段 |
| 私域引流 | 需要高粘性受众 | 任何阶段 |
| 账号出售/孵化 | 需要规模化能力 | 成熟期 |

### 变现时机判断

- 过早变现：粉丝信任未建立，转化率低且伤害账号
- 过晚变现：错过最佳窗口期
- 最佳时机信号：粉丝开始主动问「在哪买」「有没有课」
