# Seedance 2.0 提示词十条心法

> 从官方教程 + 实测案例中提炼的核心原则

## 基础七法

### 1. 时间轴分段，逐秒编排
把 15 秒拆成 3-5 个时间段，每段写清「画面 + 镜头 + 声音」。

**差**：`一个女生在街上走然后车开过来`
**好**：
```
0-3s：镜头小幅拉远并跟随女主移动，风吹拂裙摆，女主走在19世纪的伦敦大街上
3-8s：右边街道驶来一辆蒸汽机车，快速驶过女主身旁，风将裙摆吹起
8-12s：女主震惊赶忙用双手向下捂住裙摆
背景音效：走路声、人群声、汽车声
```

### 2. @引用明确，角色分工清晰
每个素材必须标清用途：

| 用途 | 写法 |
|------|------|
| 人物外观 | `参考@图片1的男人形象` |
| 场景/首帧 | `@图片1作为首帧画面` |
| 运镜/动作 | `完全参考@视频1的所有运镜效果` |
| 特效复刻 | `参考@视频1的粒子特效和材质` |
| 音乐/音效 | `背景音乐参考@视频1` |

### 3. 专业镜头语言

| 类别 | 关键词 |
|------|--------|
| 景别 | 特写、中景、全景、超近景 |
| 运镜 | 推近、拉远、环绕、跟拍、摇镜、一镜到底 |
| 角度 | 俯拍、仰拍、低机位、第一人称主观视角、鱼眼镜头 |
| 节奏 | 快速硬切、慢推、急停、定格慢放 |
| 特殊 | 希区柯克变焦、遮挡擦镜转场、无缝渐变转场 |

**关键**：用「镜头做什么」而非「画面看到什么」来描述。

### 4. 动作+表情+台词，三位一体
- 身体动作：具体到手指、脚步、衣摆
- 表情变化：从什么情绪→转为什么情绪
- 台词对白：写出具体说的话

**差**：`男人开心回家`
**好**：`男人深呼吸，调整情绪，收起负面情绪，变得轻松，然后特写翻找出钥匙，插入门锁`

### 5. 声音设计是第二画面
- 环境音：雨滴声、冰块碰杯声、衣料摩擦声
- 音乐风格：紧张刺激、轻快搞笑、低沉宫廷弦乐
- 台词语气：哽咽怒吼、冷漠反杀、调侃笑声
- 音效卡点：切换点的短促低频"砰"

### 6. 正面指令为主，负面指令兜底
先写「要什么」，再补「不要什么」：
- 正面：`动作流畅，保持人物一致性，整体风格偏写实`
- 负面：`无闪烁、无错位、不要出现字幕`

### 7. 参考 > 描述，组合 > 单一
能参考就别纯靠文字：
- 想要某运镜？→ 上传参考视频 + `参考@视频1的运镜`
- 想要某动作？→ 上传动作视频 + `参考@视频1的舞蹈动作`
- 想要某音乐？→ 上传音频文件

**黄金公式**：人物图(外貌) + 场景图(环境) + 参考视频(运镜/动作) + 文字prompt(细节)

## 进阶三法

### 8. 区分「参考」与「编辑」
- **参考** = 创造新内容：`参考@视频1的运镜方式`
- **编辑** = 修改已有内容：`将@视频1中的女生换成戏曲花旦`

### 9. 分源拆解
运镜、动作、情绪、声音可以从不同素材各取所需：
```
运镜 ← @视频1
动作 ← @视频2
人物外观 ← @图片1
场景 ← @图片2、@图片3
音乐 ← @音频1
```

限定参考范围：
- `完全参考所有运镜效果` → 全量
- `只参考形象` → 限定外观
- `音色参考@视频1` → 只取声音

### 10. 分镜图工作流

**空间路径法**：多张图代表镜头经过的不同位置
```
@图片1-5，一镜到底的追踪镜头，从街头跟随跑步者上楼梯、穿过走廊、进入屋顶
```

**时间分镜法**：多张图对应不同时间段的画面
```
0-3s：参考分镜图1... 3-6s：参考分镜图2... 6-8s：参考分镜图3...
```

**音乐卡点法**：多张图 + 音乐视频 = 画面跟节奏切
```
@图片1-7根据@视频中的画面关键帧的位置和整体节奏进行卡点
```
