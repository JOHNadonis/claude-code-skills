## AD 电商广告

### 产品360旋转展示
```
图1中的[产品名],360度高速旋转2圈后,突然停住蓄力分裂成了3个部分进行展示。随后分解后的[产品名]的上中下三部分快速向内旋转合成，一罐完整的[产品名]，3D渲染产品展示特效,动感产品特效展示
```

### 产品分解展示（汉堡/多层产品）
```
将参考图进行一个拆解，镜头保持静止，[产品]悬浮在空中，开始旋转，食材轻柔而精准地分离，保持形状和比例，动作流畅，无任何额外效果，[逐层描述每个部件]都缓缓下落并完美地组合成一个完整的[产品]。在此过程中，[质感细节描述]，保持着极致诱人的美感。
```

### 角色带货（虚拟IP/古装人设）
```
请根据参考图生成一段直播带货的视频：
0-2s 图中的人物[动作描述]，口中说道："[台词1]"
2-5s 他[与产品互动的动作]，面露惊喜，口中说道："[台词2-卖点]"
5-8s 他[使用产品的动作]，抬眼看向镜头，语气轻快说道："[台词3-场景]"
8-10s 他手持[产品]正对镜头，[推荐手势]，说道："[台词4-CTA]"
音效：[产品相关音效] + 结尾[风格提示音]
```

### 模仿参考广告片 + 替换产品
```
参考视频1的[具体风格：剪辑风格/创意/运镜转场效果]，
将视频1中的[原产品]替换成图片1的[新产品]，
并修改视频内容说明和字幕效果，
使其重点突出[新产品]的产品介绍，
保持图片1产品一致性，以图片1做为主体
```

### 产品运镜卡点
```
参考视频中的运镜方式和背景音乐，生成图片1产品跟随运镜卡点的15秒的视频
```

### 第一人称手作体验（食品/饮品）
```
[X]秒第一人称视角[产品类型]宣传广告，[品牌名]「[产品名]」，清新通透质感，柔调自然光，适配抖音/视频号信息流，画面聚焦手部动作与[产品]细节；
0-2秒：第一人称特写，你的手[制作动作1]，[音效描述]，背景音：「[文案1]」
2-4秒：快速切镜卡点，你的手[制作动作2]，[视觉效果描述]，背景音：「[文案2]」
4-6秒：第一人称成品特写，镜头拉近怼拍[成品细节]，背景音：「[文案3]」
6-8秒：第一人称手持举杯/展示，[品牌标识清晰可见]，说：「[CTA]」
```
