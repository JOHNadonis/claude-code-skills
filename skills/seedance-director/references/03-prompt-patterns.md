参考视频中的快速运镜方式

### 动作引用

### 特效引用

## 高频镜头语言句式

### 推近/拉远
```
镜头缓慢推进
推近到屏幕的特写
镜头向咖啡豆推进至画面黑屏
镜头拉远展示酒馆全景
镜头推远回到机舱内
```

### 环绕
```
几个环绕镜头展示电梯内视角
及近的环绕镜头
参考@视频1里急速的左右环绕运镜展示场景
```

### 跟拍
```
镜头跟随在他身后
镜头跟随黑衣男子快速逃亡
镜头全景跟随
镜头正面跟拍穿着红风衣的女特工向前走
继续跟拍主角奔跑的侧面视角
```

### 摇镜
```
镜头快速向右边摇动
镜头再右摇90度拍摄@图片4的分叉路口
急停后右摇180度，怼脸拍摄主角正面
快速摇镜头，转向对面特写
镜头迅速向下摇
```

### 特殊镜头
```
主角在惊恐时希区柯克变焦
鱼眼镜头效果
第一人称主观视角快推镜头
仰拍拉远定格慢放
低角度仰拍

```

## 编辑操作句式

### 替换
```
将@视频1中的女生换成戏曲花旦

将视频1女人发型变成红色长发

```

### 延长
```

```

```

```

```

```

## 约束/质量控制句式

### 一致性
```
保持人物的正脸外貌、发型、五官比例和服装不变
保持@图片1产品一致性
不遮挡面部关键特征
人物主体保持稳定
```

### 画面质量
```
画面连贯流畅，无闪烁、无错位、无风格偏移
动作流畅，无任何额外效果
整体风格偏写实
动作流畅无卡顿
```

### 禁止项
```
不要出现字幕
不要出现切镜
全程不要切镜头
禁止文字logo水印
不要出现文字或标志
不要出现人物（在需要空镜的部分）
```

## Prompt 组装检查清单

写完 prompt 后逐项检查：

- [ ] 每个 @素材 都标注了用途？
- [ ] 有时间段划分？（至少2-3段）
- [ ] 镜头语言具体？（不是泛泛的"好看的画面"）
- [ ] 人物有具体动作？（不是"站着"）
- [ ] 有声音/音效描述？
- [ ] 有情绪/表情变化？（不是全程同一表情）
- [ ] 正面指令为主？
- [ ] 需要的禁止项都写了？
- [ ] 时长与内容量匹配？（15秒别塞10个镜头）
- [ ] 编辑 vs 参考 写清楚了？
