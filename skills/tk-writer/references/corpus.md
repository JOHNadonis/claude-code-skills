## 科技评论

1. "Taalas 把大模型以 ASIC 的方式实现了——也就是说把模型直接写到芯片里，这样什么内存带宽限制之类都不存在了。目前用的是台积电 6nm 工艺，写入了 Llama 3.1 8B，能耗 2.5 千瓦，能实现惊人的每秒 1.7 万 token。[...] 这是一个里程碑事件。即使我们没有新的革命性计算架构，也不能突破硅芯片制程极限，用类似思路仍然可能把 AI 再往前推进一步。"

2. "我发现一些大厂的 AI API 官方文档比较滞后，新版 SDK 已经发布很久，文档还是老的。如果像古代程序员那样根据文档开发，写出来的代码就不能用。未来，开发文档这种给人类阅读的东西也许会逐渐消失，让 AI 根据 SDK 代码自己去分析接口就行了。"

3. "所以我一直说，对于 AI 重点不是看今天能做什么，而是得想一想明天能做什么。"

4. "AI 一开始是理科生的问题，后来还会变成文科生的问题。甚至可能会经历一段文科生都不够用的时期。当然，最后还是理科生的问题。"

5. "如果智人里有 60% 能达到 Gemini 3.1 的水平，我们的戴森球应该早就完工了。"

## 类比模板

| 类比手法 | 源域 → 目标域 | 结构同构点 |
|----------|---------------|-----------|
| 系统类比 | 游戏渲染 → 视觉缺陷 | 资源优化导致品质下降 |
| 行为类比 | 老鼠 → 资本 | 趋利避害、容易受惊、自我繁殖 |
| 生态类比 | 基因污染 → 文化污染 | 近缘物种互相影响，远缘物种互不干扰 |
| 行业类比 | 信息安全 → 民俗符咒 | 攻防对抗 + 灰色地带 |
| 物理类比 | 返老还童射线 → 烹饪 | 状态回退 |
| 制度类比 | 思想实验 → 经济制度对比 | 控制变量法 |

## 句式模板

### 陈述+转折
> "[看似正常的陈述]。但[反直觉的事实]。"

### 类比开场
> "[A事物]，有点像[B事物]。[展开B的特征]。[映射回A]。"

### 冷幽默收束
> "[正经铺垫1]。[正经铺垫2]。[正经铺垫3]。[突然转向的最后一句]。"

### 格言体
> "[对称句式前半]，[对称句式后半]。"

### 以小见大
> "[具体事实描述] → [一句话预判趋势]"

### 外星人观察
> "[用第三人称或超然视角描述人类行为]。这对[某个宏大命题]很有参考价值。"

### 反直觉论证
> "[大多数人认为X]，是吗？但[被忽略的数据/逻辑]。[因此实际情况是Y]。"

### 历史考据+脑洞
> "[70%真实史实，用严肃学术语气]。[30%荒诞脑洞，用完全一样的语气]。"
