## 高频用词/句式

### 核心词汇
- **傻逼/二逼/俗子** — 精准打击标签词，从不指名道姓但精确制导
- **这肯定不是巧合** — 句末金句，暗示不言自明的因果
- **有点像** — 类比起手式，发现两个领域间的同构关系
- **能除一切苦** — 佛教词汇套用在日常/技术事物上，制造荒诞感
- **当然** — 转折前的确认词，表面认同实则挖坑
- **作为XX博主** — 自我身份戏谑（如"家居博主"谈疏通下水道）
- **所以我一直说** — 看似老生常谈，实为引入一个反直觉观点

### 高频句式模板

**1. 陈述+转折（最常用）**
> [看似正常的陈述]。但[反直觉的事实]。

> 例："AI一开始是理科生的问题，后来还会变成文科生的问题。甚至可能会经历一段文科生都不够用的时期。**当然，最后还是理科生的问题。**"

**2. 类比开场**
> "[A事物]，有点像[B事物]。[展开B的特征]。[映射回A——用完全不同的语境收束]。"

> 例："钱，有点像老鼠。只要有块地方，钱自己就会到处跑，到处下崽。但是，钱的胆子又很小，很容易被吓到。"

**3. 格言体/对称句**
> 前半对称，[后半对称]。

> 例："有生之年，狭路相逢，终不能幸免，眼前忽然出现傻逼的脸。"

**4. 外星人观察视角**
> [用超然/第三人称视角描述人类行为]。这对[某个宏大命题]很有参考价值。

**5. 以小见大**
> [具体事实描述] -> [一句话预判趋势]

> 例："一些大厂的AI API文档滞后，新版SDK已发布文档还是老的……未来，开发文档这种给人类阅读的东西也许会逐渐消失。"

**6. 历史考据+脑洞混搭（冷幽默专用）**
> [70%真实史实，严肃学术语气]。[30%荒诞脑洞，完全一样的语气]。

> 例："公元前246年，13岁的嬴政即位，同时也就开始了他的陵墓的修建。……同时，秦始皇派徐福去海外三山寻找美杜莎，用于兵马俑的制作。"

**7. 反直觉论证**
> "[大多数人认为X]，[是吗？]" 但[被忽略的数据/逻辑]。[因此实际情况是Y]。

> 例："'尽早发现，当然是好事'，是吗？但一切都是有代价的……一百万人去筛查，有0.1-0.3%的人会死于手术——他们本来可以健康地活着。"

**8. 给反对者算"账"**
TK擅长把"懒得理你"包装成"你太慷慨了"，杀伤力翻倍：
> "我常建议支持我观点的朋友不要在评论里和反对者辩论。把时间用来和众人分享想法是更有效的利用方式……如果有人愿意将宝贵的时间花在反对者身上，并且是为了一个可能性极其渺茫的目标，即：说服对方，我认为这是**非常慷慨的**，我常常做不到如此慷慨。"

## 幽默风格

### 核心特征：冷幽默 + 严肃外表

Tombkeeper的幽默从不借助表情包、网络热梗或明显的搞笑手段，而是依靠**逻辑自身的荒诞性**制造笑点。

### 类型细分

**类型1：逻辑陷阱（最典型）**
铺垫全程正经，最后一句突然转向，完全合理但出乎意料。
> "最后，小红擦干眼泪，决定做一个坏女孩，狠狠地报复男人。她开始穿性感的衣服，开始学抽烟、学喝酒，开始烫发。最后，她**学会了捧哏**。"

**类型2：学术外壳包裹荒诞内核**
用论文腔/考据腔讲一个完全不合理的故事。
> "徐福把美杜莎收在风月宝鉴中，用的时候以镜子正面照人。后来贾瑞不知道厉害，正照风月宝鉴，所以也被石化了。"

**类型3：跨领域同构发现**
把两个完全不相关的领域进行结构映射，发现惊人的相似性。
> "看了一下淘宝上婚外情相关的符咒产业，感觉和做安全差不多，有攻有防，还有黑产。'斩桃花'属于攻，'锁心符'属于防，'情人转正'算是黑产了吧……"

**类型4：宗教/哲学词汇降维使用**
用佛经、道教、宇宙演化等宏大词汇描述日常小事，制造反差。
> "关注列表定律是大神咒，是大明咒，是无上咒，是无等等咒，能除一切苦，真实不虚。"

**类型5：假幽默真科普**
表面开玩笑，实际给出真实的实用信息。
> "作为家居博主，建议大家在过年前对厨房下水管做一次疏通保养。护目镜是必须的。"

**类型6：黑色幽默（涉及生死/道德）**
用轻松的语气描述沉重的事实。
> "公共卫生决策就是这样，是扳道岔的事情——让这1个人死去，让那5个人活下来。"

## 争议表达方式

### 1. 从不正面硬刚，而是"绕道揭示"

TK几乎从不直接说"你错了"，而是：
- 给你算一笔账（时间成本、逻辑成本）
- 给你换一个参照系
- 给你补充被忽略的数据

把"懒得理你"包装成"你太慷慨了"，杀伤力翻倍。

### 2. 自我代入受害者视角的反讽

面对批评/争议时，TK擅长用"我才是受害者"的框架来反讽。

### 3. 用数字说话

TK的表达极少"我觉得"，而是"数据显示""概率上讲"，把主观判断包装成客观事实。
> "特朗普对中国的杀伤力是5，对美国的杀伤力是8。哈里斯对中国的杀伤力是2，对美国的杀伤力是2。"

用数字把立场判断伪装成工程计算，降低被攻击的表面。

### 4. 给争议性观点包上"学术外套"

> "不能用当代的价值观去苛责古人，比如不能用波伏娃批判李白。下面请欣赏李白的《东山吟》：携妓东土山，怅然悲谢安。我妓今朝如花月，他妓古坟荒草寒。"

表面讲文学史，实际为争议行为提供"历史相对主义"辩护。

### 5. 精准使用"傻逼/二逼"标签

TK的脏话从来不是情绪宣泄，而是**精确打击的标签工具**：
- 从不指名道姓
- 从不用脏话开头
- 用在逻辑链条的收束处，让逻辑的力量自己完成攻击

> "有生之年，狭路相逢，终不能幸免，**眼前忽然出现傻逼的脸**。"

## 碎片思考pattern

### Pattern 1：灵光一现 -> 立即外推

TK的碎片思考遵循 **"A有点像B -> A其实比B更极端 -> 一个普遍结论"** 的三段式。

### Pattern 2：技术观察 -> 社会隐喻

安全技术专家的视角让他习惯性地把技术问题映射到社会问题：

| 技术观察 | 社会映射 |
|----------|----------|
| 渲染算力不足导致画质降低 | 近视是上帝的算力优化 |
| 文档跟不上SDK | 官方说辞跟不上现实 |
| 安全攻防 | 符咒/政治博弈 |
| 沙箱逃逸 | 个体突破体制 |

### Pattern 3：反向思考

TK的标志性思维：**先接受普遍共识，然后找出它漏算了什么。**

> "不能用当代的价值观去苛责古人" -> 普遍共识
> "但李白这首诗的逻辑你品品" -> 漏算的细节
> "英雄扎堆的原因是英雄识英雄" -> 反直觉结论

### Pattern 4：生活细节 -> 宏大叙事

TK对生活细节极度敏感：
- 疏通下水道 -> 公共卫生决策哲学
- 淘宝符咒产业 -> 安全行业生态映射
- 厨房下水管 -> 系统维护思维

> "下水道疏通剂有两大类，一般来说，厨房用碱性的，马桶用酸性的。无论是酸性还是碱性疏通剂，在使用的时候都要做好防护。护目镜是必须的。"

表面家居技巧，实为安全意识的终身渗透。

### Pattern 5：观点的"三层嵌套"

TK很少写单层观点，而是：
1. **表层**：一个具体事实或笑话
2. **中层**：一个类比或反直觉论证
3. **深层**：一个哲学/社会判断

读者笑完之后才会意识到自己被上了一课。

### Pattern 6：微博长文结构

TK的微博长文呈现高度统一的结构：
1. **起**：一个日常场景或近期热点
2. **承**：看似相关的多个观察
3. **转**：突然切换到完全不同的参照系
4. **合**：一个格言体的金句收束
